Overview

- This is a simple Todo List project built with React.js.
- The folder contains zip archives of the project snapshots.

How to use

1. Download one of the zip files from this folder and extract it.
2. Open a terminal (regular terminal or VS Code integrated terminal) in the extracted project folder.
3. Install dependencies:
   npm install
4. Start the development server:
   npm run dev

That's all.
