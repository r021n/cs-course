Mengaplikasikan Semua Pengetahuan untuk Mencapai Tujuan

Brando kini telah menguasai semua konsep dasar yang ia butuhkan. Ia memiliki pondasi yang kuat: variabel dan tipe data untuk menyimpan informasi, operator untuk memanipulasinya, input/output untuk berinteraksi, percabangan untuk membuat keputusan, perulangan untuk otomasi, fungsi untuk organisasi, dan struktur data (list, dictionary, tuple, set) untuk mengelola data yang kompleks. Sekarang, saatnya ia menggabungkan semua ini untuk menciptakan program impiannya.

Tujuannya adalah membuat program yang bisa mencatat penjualan, menghitung laba, dan memberikan ringkasan mingguan. Ia mulai merancang kode utamanya dengan menggunakan semua konsep yang sudah ia pelajari. 

# Mendefinisikan harga dan stok kue menggunakan dictionary
daftar_kue = {
    "Kue Cokelat": {"harga": 12000, "stok": 50},
    "Kue Keju": {"harga": 15000, "stok": 40},
    "Kue Pandan": {"harga": 13500, "stok": 60}
}

# Mendefinisikan biaya produksi per kue
biaya_produksi = {
    "Kue Cokelat": 8000,
    "Kue Keju": 10000,
    "Kue Pandan": 9500
}

# List untuk menyimpan catatan penjualan harian
catatan_penjualan_mingguan = []

# Fungsi untuk mencatat penjualan
def catat_penjualan(kue_terjual, jumlah):
    if kue_terjual in daftar_kue:
        if jumlah <= daftar_kue[kue_terjual]["stok"]:
            # Menghitung laba dan total
            harga_jual = daftar_kue[kue_terjual]["harga"]
            harga_pokok = biaya_produksi[kue_terjual]
            laba_per_kue = harga_jual - harga_pokok
            
            total_penjualan = jumlah * harga_jual
            total_laba = jumlah * laba_per_kue
            
            # Mengurangi stok
            daftar_kue[kue_terjual]["stok"] -= jumlah
            
            # Menyimpan data penjualan ke dalam list
            catatan_penjualan_mingguan.append({
                "nama": kue_terjual,
                "jumlah": jumlah,
                "total_penjualan": total_penjualan,
                "total_laba": total_laba
            })
            print(f"Berhasil mencatat {jumlah} {kue_terjual}.")
            return True
        else:
            print("Maaf, stok tidak mencukupi.")
            return False
    else:
        print("Maaf, kue tidak ditemukan dalam daftar.")
        return False

# Fungsi untuk menampilkan ringkasan mingguan
def tampilkan_ringkasan():
    total_laba_mingguan = 0
    total_penjualan_mingguan = 0
    
    print("\n--- Ringkasan Penjualan Mingguan ---")
    
    # Perulangan for untuk menjumlahkan semua penjualan
    for catatan in catatan_penjualan_mingguan:
        print(f"{catatan['nama']} - Terjual: {catatan['jumlah']} - Total Penjualan: {catatan['total_penjualan']} - Laba: {catatan['total_laba']}")
        total_laba_mingguan += catatan["total_laba"]
        total_penjualan_mingguan += catatan["total_penjualan"]
        
    print("-" * 30)
    print(f"Total Penjualan Mingguan: {total_penjualan_mingguan} rupiah")
    print(f"Total Laba Mingguan: {total_laba_mingguan} rupiah")
    print("-" * 30)
    
# Perulangan while untuk interaksi program
while True:
    print("\n[Pilihan] 1. Catat Penjualan | 2. Lihat Ringkasan | 3. Keluar")
    pilihan = input("Masukkan pilihan Anda: ")
    
    if pilihan == "1":
        print("\n--- Mencatat Penjualan ---")
        kue = input("Nama kue yang terjual: ")
        jumlah = int(input("Jumlah yang terjual: "))
        catat_penjualan(kue, jumlah)
    elif pilihan == "2":
        tampilkan_ringkasan()
    elif pilihan == "3":
        print("Terima kasih. Program selesai.")
        break  # Menghentikan perulangan while
    else:
        print("Pilihan tidak valid.")

Kode di atas adalah puncak dari semua yang telah Brando pelajari. Ia menggunakan dictionary yang terstruktur untuk menyimpan data kue, termasuk harga dan stok. Ia menggunakan list untuk menyimpan semua catatan penjualan sebagai entri-entri terpisah. Ia menggunakan fungsi untuk mengorganisir logika pencatatan dan ringkasan. Ia menggunakan percabangan (if-elif-else) untuk memvalidasi pilihan pengguna dan memeriksa ketersediaan stok. Dan ia menggunakan perulangan (while) untuk menjaga programnya tetap berjalan sampai ibunya selesai menggunakannya.
