Modul 1: Mendeteksi Objek dalam Sebuah Gambar

Selamat datang di petualangan pertama kita! Di modul ini, kita akan belajar cara menyuruh "detektif AI" kita (model YOLOv8) untuk menemukan objek-objek yang tersembunyi di dalam sebuah gambar. Anggap saja kita memberinya sebuah foto TKP (Tempat Kejadian Perkara), dan tugasnya adalah melaporkan semua benda penting yang ia lihat.

Prerequisite (Persiapan Sebelum Memulai)

Sebelum memanggil detektif kita, pastikan laptop atau komputermu sudah siap dengan peralatan berikut:

  1. Python Sudah Terpasang: Ini adalah bahasa yang kita gunakan untuk memberi perintah.
  2. Install Library `ultralytics`: Ini adalah "kantor" tempat detektif YOLOv8 kita bekerja. Untuk menginstallnya, buka Terminal atau Command Prompt, lalu ketik: pip install ultralytics
  3. Satu File Model YOLOv8: Unduh satu "otak" detektif. Kita akan pakai yang paling kecil dan cepat, yaitu yolov8n.pt. Library ultralytics biasanya akan mengunduh ini secara otomatis saat pertama kali digunakan, jadi kita tidak perlu memikirkannya.
  4. Satu Gambar untuk Dideteksi: Siapkan sebuah file gambar (misalnya motor_dan_mobil.jpg) yang berisi objek-objek seperti orang, mobil, motor, atau apa pun yang menarik. Simpan di folder yang sama dengan kode Python-mu nanti.

Langkah 1: Memanggil Sang Detektif (Memuat Model)

Penjelasan Langkah

Bayangkan file yolov8n.pt itu bukan sekadar file biasa. Anggaplah itu adalah sebuah "otak" buatan yang sudah dilatih dengan jutaan gambar. Otak ini sudah sangat ahli dalam mengenali pola-pola objek. Tugas pertama kita adalah memuat "otak" ini dari file ke dalam memori program kita, sehingga ia siap menerima perintah. Proses ini seperti membangunkan seorang detektif dari tidurnya dan berkata, "Hey, bangun! Ada tugas untukmu."

Kode

# Impor kelas YOLO dari "kantor"-nya
from ultralytics import YOLO

# Kita panggil dan siapkan detektif kita.
# Kita beri tahu dia untuk menggunakan "otak" bernama 'yolov8n.pt'
model = YOLO('yolov8n.pt')

Penjelasan Kode

  • from ultralytics import YOLO: Di baris ini, kita pergi ke "kantor" (ultralytics) dan memanggil kelas utama bernama YOLO. Kelas ini adalah cetak biru untuk membuat objek detektif kita.
  • model = YOLO('yolov8n.pt'): Di sini kita menciptakan "instansi" atau objek detektif kita. Kita beri nama model. Argumen 'yolov8n.pt' adalah instruksi spesifik: "Gunakan pengetahuan dari file yolov8n.pt". Jika filenya belum ada di komputermu, kode ini cukup pintar untuk mengunduhnya secara otomatis. Sekarang, variabel model berisi detektif AI kita yang sudah siap beraksi.

Langkah 2: Memberi Tugas pada Detektif (Menjalankan Prediksi)

Penjelasan Langkah

Setelah detektif kita siap, saatnya memberinya kasus pertama: sebuah gambar. Saat kita memberikan gambar ke model, proses ini disebut inferensi atau prediksi. Model akan "melihat" gambar tersebut (yang bagi komputer hanyalah sekumpulan angka-angka piksel) dan menggunakan semua pengetahuannya untuk menemukan objek di dalamnya.

Hasilnya bukanlah gambar yang sudah ada kotak-kotaknya. Ini bagian penting yang sering disalahpahami. Hasil mentahnya adalah sebuah laporan detail dalam bentuk data. Laporan ini berisi: "Aku menemukan objek A di koordinat ini, dengan tingkat keyakinan sekian persen. Aku juga menemukan objek B di koordinat itu..." dan seterusnya.

Kode

# ... (kode dari bagian sebelumnya) ...

# Detektif 'model' kita sekarang kita beri tugas untuk menganalisis gambar.
# Ganti 'motor_dan_mobil.jpg' dengan nama file gambarmu.
results = model('motor_dan_mobil.jpg')

Penjelasan Kode

  • results = model('motor_dan_mobil.jpg'): Ini adalah baris perintah utamanya. Kita memanggil detektif kita (model) dan memberinya satu kasus, yaitu file gambar 'motor_dan_mobil.jpg'. Semua hasil temuannya, dalam bentuk laporan data yang sangat terstruktur, akan disimpan di dalam variabel bernama results.

Langkah 3: Membaca Laporan Hasil Investigasi (Mengurai Hasil Deteksi)

Penjelasan Langkah

Variabel results tadi bukanlah gambar, melainkan sebuah "koper" berisi data. Kita harus membuka koper ini dan membaca laporannya satu per satu. Laporan ini sangat detail, berisi informasi tentang setiap objek yang ditemukan. Tiga informasi terpenting dari setiap temuan adalah:

  1. Bounding Box (Kotak Pembatas): Ini adalah 4 angka yang menentukan koordinat pojok kiri-atas dan pojok kanan-bawah dari kotak yang mengelilingi objek. Inilah yang nantinya kita gunakan untuk menggambar kotak.
  2. Confidence Score (Skor Keyakinan): Ini adalah angka antara 0 dan 1 yang menunjukkan seberapa yakin si detektif dengan temuannya. Skor 0.95 artinya dia 95% yakin itu adalah objek yang benar.
  3. Class (Kelas Objek): Ini adalah nama dari objek yang ditemukan, misalnya "person", "car", atau "dog".

Kode Lanjutan

# ... (kode dari bagian sebelumnya) ...

# Karena 'results' bisa berisi hasil dari beberapa gambar, kita perlu loop.
# Tapi karena kita hanya memproses satu gambar, loop ini hanya akan berjalan sekali.
for r in results:
    # Kita minta koper tadi untuk diubah menjadi gambar yang sudah digambari kotak
    # Inilah cara paling mudah untuk visualisasi!
    im_array = r.plot()

Penjelasan Kode

  • for r in results:: Variabel results didesain untuk bisa menampung hasil dari banyak gambar sekaligus. Walaupun kita hanya punya satu, cara mengaksesnya tetap menggunakan perulangan (loop). r di sini akan menjadi laporan detail untuk satu gambar kita.
  • im_array = r.plot(): Ini adalah sebuah fungsi ajaib yang disediakan oleh ultralytics. Fungsi .plot() ini membaca semua data mentah di dalam r (koordinat kotak, nama kelas, skor keyakinan), lalu secara otomatis menggambar semua itu di atas gambar aslinya. Hasilnya adalah sebuah gambar baru (dalam format data yang disebut NumPy array) yang sudah ada coretan-coretan hasil deteksinya. Gambar baru ini kita simpan di variabel im_array.

Langkah 4: Menunjukkan Bukti ke Publik (Menampilkan Gambar Hasil)

Penjelasan Langkah

Kita sudah punya gambar baru di dalam variabel im_array. Tapi saat ini, gambar itu hanya ada di dalam memori komputer. Manusia tidak bisa melihatnya. Tugas terakhir kita adalah mengambil data gambar tersebut dan menampilkannya di sebuah jendela di layar monitor kita. Untuk melakukan ini, kita butuh bantuan dari "asisten" lain, yaitu sebuah library bernama OpenCV (cv2), yang sangat ahli dalam urusan menampilkan gambar dan video.

Kode Lengkap

# Impor kelas YOLO dari "kantor"-nya
from ultralytics import YOLO
# Impor asisten untuk menampilkan gambar
import cv2

# 1. Panggil dan siapkan detektif kita
model = YOLO('yolov8n.pt')

# 2. Beri tugas pada detektif untuk menganalisis gambar
results = model('motor_dan_mobil.jpg') # Ganti dengan nama file gambarmu

# 3. Baca laporan dan minta hasilnya digambar
for r in results:
    im_array = r.plot()  # Ini menghasilkan gambar dalam format NumPy array (BGR)

    # 4. Tampilkan gambar bukti ke publik menggunakan OpenCV
    cv2.imshow('Hasil Deteksi YOLOv8', im_array)

    # Tunggu sampai ada tombol keyboard yang ditekan sebelum menutup jendela
    cv2.waitKey(0)

# Setelah selesai, tutup semua jendela yang dibuka oleh OpenCV
cv2.destroyAllWindows()

Penjelasan Kode

  • import cv2: Memanggil asisten OpenCV agar siap digunakan. Pastikan sudah terinstall (pip install opencv-python).
  • cv2.imshow('Hasil Deteksi YOLOv8', im_array): Perintah untuk si asisten. "Tolong, tampilkan sebuah jendela. Beri judul jendela itu 'Hasil Deteksi YOLOv8', dan gambar yang mau ditampilkan ada di dalam variabel im_array."
  • cv2.waitKey(0): Ini adalah baris yang sangat penting dan sering membuat bingung pemula. Komputer bekerja super cepat. Tanpa baris ini, program akan menampilkan jendela lalu langsung menutupnya dalam sepersekian detik, kita bahkan tidak akan sempat melihatnya. Perintah ini artinya: "Jeda programnya di sini. Jangan tutup jendelanya sampai pengguna menekan tombol apa saja di keyboard." Angka 0 berarti menunggu selamanya.
  • cv2.destroyAllWindows(): Setelah pengguna menekan sebuah tombol, waitKey(0) akan selesai. Perintah ini adalah langkah yang baik untuk membereskan dan menutup semua jendela yang tadi sudah kita buka.

Output yang Diharapkan

Setelah menjalankan kode lengkap di atas, sebuah jendela akan muncul di layarmu. Jendela tersebut akan menampilkan gambar motor_dan_mobil.jpg milikmu, tetapi sekarang sudah ada kotak-kotak berwarna yang mengelilingi setiap objek yang berhasil dideteksi. Di atas setiap kotak, akan ada label nama objek beserta skor keyakinannya (misalnya: car 0.89). Jendela ini akan tetap terbuka sampai kamu menekan tombol apa pun pada keyboardmu.
