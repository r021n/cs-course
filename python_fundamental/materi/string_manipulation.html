"Manipulasi String & Formatting" – Merapikan Teks di Python

Kita sudah tahu bahwa String adalah tipe data untuk menyimpan teks (deretan karakter). Dari nama, alamat, hingga pesan panjang, semuanya adalah string. Namun, jarang sekali kita menyimpan teks hanya untuk ditampilkan persis seperti itu. Seringkali, kita perlu mengubahnya, memotongnya, menggabungkannya, atau merapikannya agar terlihat lebih baik dan informatif.

Bayangkan Anda memiliki nama depan dan nama belakang secara terpisah, lalu Anda ingin menggabungkannya menjadi nama lengkap. Atau, Anda ingin menampilkan angka yang besar dengan format ribuan, atau membuat tabel sederhana dengan teks yang rapi. Di sinilah peran "Manipulasi String & Formatting". Ini adalah seni membuat teks di program Anda bekerja sesuai keinginan dan terlihat menarik.

Menggabungkan Teks: Operator +

Kita sudah melihat sekilas bagaimana operator + bisa digunakan untuk menggabungkan dua teks (string) menjadi satu. Ini seperti menyambung dua potongan tali menjadi satu tali yang lebih panjang. 

nama_depan = "Dian"
nama_belakang = "Permata"

# Menggabungkan dua string dengan spasi di tengah
nama_lengkap = nama_depan + " " + nama_belakang
print("Nama Lengkap:", nama_lengkap)

kota = "Bandung"
provinsi = "Jawa Barat"
alamat = "Tinggal di " + kota + ", " + provinsi
print(alamat)

# Ingat, jika ada angka, harus dikonversi ke string dulu!
usia = 17
pesan_usia = "Usia saya " + str(usia) + " tahun." # str(usia) mengubah 17 menjadi "17"
print(pesan_usia)

Outputnya:

Nama Lengkap: Dian Permata
Tinggal di Bandung, Jawa Barat
Usia saya 17 tahun.

Penjelasan Kode:

Operator + mengambil string di kiri dan string di kanan, lalu menggabungkannya. Kita menambahkan spasi " " di antara nama_depan dan nama_belakang agar tidak terlihat menempel. Penting untuk diingat lagi, jika Anda ingin menggabungkan teks dengan angka, Anda harus mengubah angka itu menjadi teks terlebih dahulu menggunakan str(), seperti pada str(usia). Kalau tidak, Python akan bingung dan mengeluarkan error.

Mengulang Teks: Operator *

Kadang Anda ingin mengulang sebuah teks berkali-kali. Operator * yang biasanya untuk perkalian angka, juga bisa digunakan untuk mengulang string! Anda bisa mengalikan string dengan sebuah angka (bilangan bulat positif). 

# Mengulang sebuah kata
teks_ulang = "Hai!" * 3
print("Teks diulang:", teks_ulang)

# Membuat garis pemisah sederhana
garis_pemisah = "=" * 20
print(garis_pemisah)
print("Selamat Datang")
print(garis_pemisah)

# Mengulang spasi
spasi_banyak = " " * 5
print("Ini adalah" + spasi_banyak + "teks dengan banyak spasi.")

Outputnya:

Teks diulang: Hai!Hai!Hai!
====================
Selamat Datang
====================
Ini adalah     teks dengan banyak spasi.

Penjelasan Kode:

Ketika Anda menulis "Hai!" * 3, Python akan membuat string "Hai!" muncul sebanyak 3 kali secara berurutan. Ini sangat berguna untuk membuat pola sederhana, garis pembatas, atau indentasi (jarak kosong) dengan mudah.

Mengetahui Panjang Teks: Fungsi len()

Seringkali kita perlu tahu berapa banyak karakter (termasuk spasi) yang ada dalam sebuah teks. Python punya fungsi bawaan yang disebut len() (kependekan dari "length" atau panjang). Anda cukup meletakkan string atau variabel string di dalam kurung (). 

kalimat = "Python itu mudah dipelajari."
panjang_kalimat = len(kalimat)

nama_kosong = ""
panjang_nama_kosong = len(nama_kosong)

print("Kalimat:", kalimat)
print("Panjang kalimat:", panjang_kalimat, "karakter.") # Spasi juga dihitung!

print("Panjang nama kosong:", panjang_nama_kosong, "karakter.")

Outputnya:

Kalimat: Python itu mudah dipelajari.
Panjang kalimat: 28 karakter.
Panjang nama kosong: 0 karakter.

Penjelasan Kode:

Fungsi len(kalimat) menghitung setiap karakter dalam string kalimat, termasuk spasi dan tanda baca, menghasilkan angka 28. String kosong "" memiliki panjang 0. len() sangat berguna, misalnya, untuk membatasi jumlah karakter yang bisa dimasukkan pengguna atau untuk mengatur tampilan agar rapi.

Mengambil Bagian Teks: Mengakses Karakter Individu (Indexing)

Sebuah string sebenarnya adalah kumpulan karakter yang berurutan. Kita bisa mengambil satu karakter tertentu dari sebuah string dengan mengetahui "posisinya" atau yang disebut Indeks. Dalam Python (dan banyak bahasa pemrograman lain), penghitungan indeks selalu dimulai dari nol (0), bukan satu.

  • nama[0] berarti karakter pertama.
  • nama[1] berarti karakter kedua, dst.

Anda juga bisa menggunakan indeks negatif untuk menghitung dari belakang:

  • nama[-1] berarti karakter terakhir.
  • nama[-2] berarti karakter kedua dari belakang, dst. 
huruf_vokal = "AEIOU"

print("Huruf pertama:", huruf_vokal[0])
print("Huruf kedua:", huruf_vokal[1])
print("Huruf terakhir:", huruf_vokal[4]) # Karena total 5 huruf, indeks terakhir 4
print("Huruf terakhir (dari belakang):", huruf_vokal[-1])
print("Huruf kedua dari belakang:", huruf_vokal[-2])

nama_lengkap_siswa = "Rudi Hartono"
print("Inisial depan:", nama_lengkap_siswa[0])
print("Inisial belakang:", nama_lengkap_siswa[5]) # Karakter 'H' ada di indeks 5 (setelah spasi)

# print(huruf_vokal[5]) # Ini akan error! Indeks di luar batas

Outputnya:

Huruf pertama: A
Huruf kedua: E
Huruf terakhir: U
Huruf terakhir (dari belakang): U
Huruf kedua dari belakang: O
Inisial depan: R
Inisial belakang: H

Penjelasan Kode:

huruf_vokal[0] memberikan 'A' karena 'A' adalah karakter pertama (indeks 0). huruf_vokal[-1] juga memberikan 'U' karena itu adalah karakter terakhir. Teknik ini berguna untuk mengambil inisial, atau memproses karakter satu per satu. Anda harus berhati-hati agar tidak mencoba mengakses indeks yang tidak ada, karena itu akan menyebabkan error. (Baris print(huruf_vokal[5]) sengaja dikomentari karena akan menyebabkan error IndexError.)

Mengambil Sebagian Teks: Memotong String (Slicing)

Selain mengambil satu karakter, kita juga bisa mengambil "potongan" atau sebagian dari string yang lebih panjang. Ini disebut Slicing. Anda perlu menentukan titik awal dan titik akhir (opsional, bisa juga langkah-langkahnya). Sintaksnya adalah [mulai:akhir:langkah].

  • mulai: Indeks di mana potongan dimulai (termasuk karakter di indeks ini). Jika dikosongkan, mulai dari awal.
  • akhir: Indeks di mana potongan berakhir (karakter di indeks ini TIDAK termasuk). Jika dikosongkan, sampai akhir.
  • langkah: Seberapa jauh kita melangkah (misalnya, ambil setiap 2 karakter). Jika dikosongkan, defaultnya 1. 
teks_panjang = "Belajar Python itu sangat menyenangkan!"

# Mengambil dari indeks 0 sampai sebelum indeks 7
kata_pertama = teks_panjang[0:7]
print("Kata pertama:", kata_pertama)

# Mengambil dari indeks 8 sampai akhir
sisa_kalimat = teks_panjang[8:]
print("Sisa kalimat:", sisa_kalimat)

# Mengambil dari awal sampai sebelum indeks 7
kata_lain = teks_panjang[:7]
print("Kata lain:", kata_lain)

# Mengambil 5 karakter terakhir
lima_terakhir = teks_panjang[-5:]
print("Lima karakter terakhir:", lima_terakhir)

# Membalikkan string (trik slicing khusus!)
teks_terbalik = teks_panjang[::-1]
print("Teks terbalik:", teks_terbalik)

# Mengambil setiap 2 karakter
setiap_dua = teks_panjang[::2]
print("Setiap dua karakter:", setiap_dua)

Outputnya:

Kata pertama: Belajar
Sisa kalimat: Python itu sangat menyenangkan!
Kata lain: Belajar
Lima karakter terakhir: gkan!
Teks terbalik: !nakgnaneynem tagnas uti nohtyP rajaleB
Setiap dua karakter: BljrPto t agtmneaga!

Penjelasan Kode:

Slicing sangat fleksibel. teks_panjang[0:7] mengambil karakter dari indeks 0 hingga indeks 6 (ingat, indeks akhir tidak termasuk). teks_panjang[8:] berarti ambil dari indeks 8 sampai akhir. teks_panjang[:7] berarti ambil dari awal sampai sebelum indeks 7.

Trik [::-1] sangat menarik: ini berarti mulai dari awal hingga akhir, dengan langkah -1 (mundur). Hasilnya adalah string yang terbalik! Sedangkan [::2] berarti mulai dari awal hingga akhir, mengambil setiap karakter kedua.

Aksi Keren String: Menggunakan "Metode String"

String di Python itu bukan cuma kumpulan karakter, tapi juga punya "kemampuan" atau "aksi" bawaan yang bisa mereka lakukan. Aksi-aksi ini disebut Metode String. Anda bisa memanggil metode dengan menulis nama variabel string, lalu titik (.), lalu nama metodenya, diikuti tanda kurung (). Ini seperti Anda meminta variabel string untuk "melakukan sesuatu".

Berikut beberapa metode string yang sangat berguna:

  • lower(): Mengubah semua huruf menjadi huruf kecil.
  • upper(): Mengubah semua huruf menjadi huruf kapital.
  • strip(): Menghilangkan spasi berlebihan (di awal atau akhir string).
  • replace(lama, baru): Mengganti semua kemunculan teks lama dengan teks baru.
  • startswith(awalan): Memeriksa apakah string dimulai dengan teks awalan tertentu (hasilnya True atau False).
  • endswith(akhiran): Memeriksa apakah string diakhiri dengan teks akhiran tertentu (hasilnya True atau False).
  • count(teks): Menghitung berapa kali sebuah teks atau karakter muncul di dalam string.
  • find(teks): Mencari posisi (indeks) pertama kali sebuah teks muncul. Jika tidak ditemukan, hasilnya -1. 
judul_artikel = "   Belajar PYTHON itu MUDAH!   "

# Mengubah huruf
judul_kecil = judul_artikel.lower()
judul_besar = judul_artikel.upper()
print("Judul kecil:", judul_kecil)
print("Judul besar:", judul_besar)

# Menghilangkan spasi berlebih
judul_bersih = judul_artikel.strip()
print("Judul bersih:", judul_bersih)

# Mengganti kata
judul_baru = judul_bersih.replace("MUDAH", "MENYENANGKAN")
print("Judul baru:", judul_baru)

# Memeriksa awalan dan akhiran
apakah_dimulai_belajar = judul_bersih.startswith("Belajar")
apakah_diakhiri_mudah = judul_bersih.endswith("MUDAH!") # Perhatikan tanda seru!
print("Dimulai dengan 'Belajar'?", apakah_dimulai_belajar)
print("Diakhiri dengan 'MUDAH!'?", apakah_diakhiri_mudah)

# Menghitung kemunculan huruf 'u'
jumlah_u = judul_bersih.count('u')
print("Jumlah huruf 'u':", jumlah_u)

# Mencari posisi kata 'itu'
posisi_itu = judul_bersih.find("itu")
print("Kata 'itu' ditemukan di indeks:", posisi_itu)
posisi_tidak_ada = judul_bersih.find("sulit") # Kata 'sulit' tidak ada
print("Kata 'sulit' ditemukan di indeks:", posisi_tidak_ada)

Outputnya:

Judul kecil:    belajar python itu mudah!   
Judul besar:    BELAJAR PYTHON ITU MUDAH!   
Judul bersih: Belajar PYTHON itu MUDAH!
Judul baru: Belajar PYTHON itu MENYENANGKAN!
Dimulai dengan 'Belajar'? True
Diakhiri dengan 'MUDAH!'? True
Jumlah huruf 'u': 2
Kata 'itu' ditemukan di indeks: 15
Kata 'sulit' ditemukan di indeks: -1

Penjelasan Kode:

Setiap metode ini melakukan tugas spesifik pada string dan mengembalikan string baru (atau nilai Boolean/angka). Penting untuk diingat bahwa metode ini tidak mengubah string asli, tetapi mengembalikan string yang sudah diubah. Jadi, judul_artikel tetap sama, sedangkan judul_kecil adalah hasil dari lower().

strip() sangat berguna untuk membersihkan input pengguna yang mungkin memiliki spasi tidak sengaja di awal atau akhir. replace() bisa sangat ampuh untuk melakukan koreksi teks secara otomatis. Metode startswith() dan endswith() sangat bagus untuk memvalidasi input atau mengidentifikasi jenis teks tertentu. find() sangat berguna untuk mencari keberadaan teks dan mengetahui lokasinya.

Memformat String: f-strings (Cara Paling Modern dan Canggih)

Meskipun operator + bisa menggabungkan string, itu bisa jadi rumit jika Anda punya banyak variabel dan teks yang ingin digabungkan. Python 3.6 ke atas memperkenalkan cara yang jauh lebih elegan dan kuat untuk memformat string, yang disebut f-strings (kependekan dari "formatted string literals").

Dengan f-strings, Anda cukup menulis huruf f di depan tanda kutip string, lalu Anda bisa langsung menulis nama variabel atau bahkan melakukan perhitungan di dalam kurung kurawal {} di dalam string tersebut. Python akan secara otomatis mengganti {} dengan nilai yang sesuai. 

# Contoh menggunakan f-strings

nama_produk = "Buku Sejarah"
harga_satuan = 75000
stok = 12

# Menggabungkan teks dan angka dengan mudah
pesan_produk = f"Produk: {nama_produk}, Harga: Rp{harga_satuan}, Stok Tersedia: {stok} unit."
print(pesan_produk)

# Bisa melakukan perhitungan langsung di dalam f-string
total_harga = harga_satuan * stok
ringkasan_pesanan = f"Anda memesan {stok} buah {nama_produk} dengan total harga Rp{total_harga}."
print(ringkasan_pesanan)

# Memformat angka desimal (akan dibahas lebih lanjut di formatting lanjutan)
pi_value = 3.1415926535
formatted_pi = f"Nilai PI yang dibulatkan: {pi_value:.2f}" # .2f artinya 2 angka di belakang koma
print(formatted_pi)

# Menampilkan Boolean atau tipe data lain
apakah_aktif = True
status = f"Status akun: {apakah_aktif}"
print(status)

Outputnya:

Produk: Buku Sejarah, Harga: Rp75000, Stok Tersedia: 12 unit.
Anda memesan 12 buah Buku Sejarah dengan total harga Rp900000.
Nilai PI yang dibulatkan: 3.14
Status akun: True

Penjelasan Kode:

Kekuatan f-strings terletak pada kemampuannya untuk "memasukkan" nilai variabel atau hasil ekspresi langsung ke dalam string dengan sangat mudah. Bandingkan f"Produk: {nama_produk}" dengan "Produk: " + nama_produk, tentu jauh lebih ringkas yang pertama, bukan?

Anda bisa memasukkan variabel apapun ke dalam {}. Python akan secara otomatis mengubahnya menjadi teks. Bahkan, Anda bisa melakukan operasi matematika di dalamnya, seperti {harga_satuan * stok}. Fitur formatting seperti {pi_value:.2f} (yang membulatkan angka pi_value menjadi dua angka di belakang koma) menunjukkan betapa canggihnya f-strings.

Dengan memahami manipulasi string dan teknik formatting, Anda sekarang memiliki kendali penuh atas bagaimana teks ditampilkan di program Anda. Ini adalah kemampuan yang sangat penting untuk menciptakan program yang tidak hanya berfungsi dengan baik, tetapi juga mudah digunakan dan dipahami oleh orang lain!
