Python Fundamental

"Tipe Data" – Jenis-Jenis Informasi yang Dimengerti Python

Bayangkan Anda memiliki sebuah kotak berisi pensil,dan sebuah kotak berisi permen. Anda tidak akan mencoba menulis dengan permen, atau memakan pensil. Setiap "barang" memiliki jenisnya sendiri dan cara kita memperlakukannya juga berbeda.

Sama seperti itu, dalam dunia pemrograman, komputer perlu tahu jenis informasi apa yang sedang ia tangani. Apakah itu angka, teks, atau sesuatu yang lain? Inilah yang disebut Tipe Data. Python sangat cerdas dalam mengenali berbagai tipe data secara otomatis. Memahami tipe data membantu kita memastikan program melakukan operasi yang benar pada jenis informasi yang tepat.

Angka Utuh: Tipe Data Integer (Bilangan Bulat)

Ketika kita berurusan dengan bilangan bulat – yaitu angka yang tidak memiliki tanda koma atau bagian pecahan – Python menyebutnya sebagai Integer (sering disingkat int). Ini adalah tipe data yang paling umum untuk menghitung jumlah barang, umur, tahun, dan sejenisnya. 

# Ini adalah variabel yang menyimpan bilangan bulat
jumlah_apel = 10
tahun_lahir = 2007
skor_ujian = 95

print("Jumlah apel yang tersedia:", jumlah_apel)
print("Anda lahir di tahun:", tahun_lahir)
print("Skor ujian Anda:", skor_ujian)

Outputnya:

Jumlah apel yang tersedia: 10
Anda lahir di tahun: 2007
Skor ujian Anda: 95

Penjelasan Kode:

Dalam contoh di atas, angka 10, 2007, dan 95 semuanya adalah bilangan bulat. Ketika kita menyimpannya ke dalam variabel seperti jumlah_apel, Python secara otomatis mengenali bahwa ini adalah tipe data Integer. Python akan memproses angka-angka ini sebagai angka utuh, yang cocok untuk operasi matematika dasar.

Angka Pecahan: Tipe Data Float (Bilangan Desimal)

Jika Anda membutuhkan angka yang memiliki bagian desimal (angka di belakang koma, atau lebih tepatnya di belakang titik dalam penulisan standar internasional), Python memiliki tipe data khusus yang disebut Float. Tipe ini sangat berguna untuk mengukur sesuatu yang tidak selalu bulat, seperti tinggi badan, berat, suhu, atau harga. 

# Ini adalah variabel yang menyimpan bilangan desimal
tinggi_badan_meter = 1.75
suhu_ruangan = 25.5
harga_barang = 15500.50 # Gunakan titik untuk desimal, bukan koma

print("Tinggi badan saya:", tinggi_badan_meter, "meter")
print("Suhu ruangan saat ini:", suhu_ruangan, "derajat Celsius")
print("Harga satuannya adalah Rp", harga_barang)

Outputnya:

Tinggi badan saya: 1.75 meter
Suhu ruangan saat ini: 25.5 derajat Celsius
Harga satuannya adalah Rp 15500.5

Penjelasan Kode:

Perhatikan angka 1.75, 25.5, dan 15500.50. Mereka semua memiliki bagian desimal. Python secara otomatis mengidentifikasinya sebagai tipe data Float. Penting diingat bahwa dalam Python (dan kebanyakan bahasa pemrograman), tanda titik (.) digunakan sebagai pemisah desimal, bukan koma. Jadi, 15.500,50 yang biasa kita tulis dalam rupiah akan menjadi 15500.50 di Python.

Teks: Tipe Data String (Deretan Karakter)

Ketika Anda ingin menyimpan nama, alamat, kalimat, atau bahkan kode pos yang mungkin mengandung angka tetapi Anda tidak akan menghitungnya (misalnya "40123"), Python menggunakan tipe data String (sering disingkat str). String adalah urutan karakter apa pun yang diapit oleh tanda kutip, baik itu kutip tunggal (') atau kutip ganda ("). 

# Ini adalah variabel yang menyimpan teks (string)
nama_sekolah = "SMA Negeri 1 Jaya"
pesan_motivasi = 'Jangan menyerah, teruslah belajar!'
kode_pos = "40123" # Meskipun angka, ini adalah teks karena diapit kutip

print("Nama sekolah saya:", nama_sekolah)
print("Pesan hari ini:", pesan_motivasi)
print("Kode pos daerah saya:", kode_pos)

Outputnya:

Nama sekolah saya: SMA Negeri 1 Jaya
Pesan hari ini: Jangan menyerah, teruslah belajar!
Kode pos daerah saya: 40123

Penjelasan Kode:

Teks "SMA Negeri 1 Jaya", 'Jangan menyerah, teruslah belajar!', dan "40123" semuanya adalah String. Python mengenali mereka sebagai teks karena diapit oleh tanda kutip. Penting untuk diingat: meskipun kode_pos berisi angka, karena diapit tanda kutip, Python memperlakukannya sebagai teks. Anda tidak bisa melakukan operasi matematika pada string ini, bahkan jika isinya angka. Mencoba menambahkan kode_pos dengan angka lain akan menghasilkan error, karena Python tidak tahu bagaimana "menambahkan" teks dengan angka.

True dan False: Tipe Data Boolean

Ada kalanya kita hanya perlu menyimpan informasi yang jawabannya hanya "ya" atau "tidak", "benar" atau "salah". Untuk situasi ini, Python memiliki tipe data Boolean (sering disingkat bool). Tipe data ini hanya memiliki dua kemungkinan nilai: True (Benar) atau False (Salah). Perhatikan bahwa huruf pertama (T dan F) harus selalu huruf kapital. 

# Ini adalah variabel Boolean
apakah_siswa_hadir = True
sudah_pulang = False
nilai_memuaskan = True

print("Apakah siswa ini hadir?", apakah_siswa_hadir)
print("Apakah pelajaran sudah selesai?", sudah_pulang)
print("Apakah nilai Anda memuaskan?", nilai_memuaskan)

Outputnya:

Apakah siswa ini hadir? True
Apakah pelajaran sudah selesai? False
Apakah nilai Anda memuaskan? True

Penjelasan Kode:

Variabel apakah_siswa_hadir, sudah_pulang, dan nilai_memuaskan semuanya menyimpan nilai Boolean. Nilai True dan False ini sangat penting dalam membuat keputusan di program Anda, yang akan kita pelajari lebih lanjut di modul Kondisional.

Mengapa Python Perlu Tahu Tipe Data?

Mungkin Anda bertanya, "Untuk apa saya pusing-pusing memikirkan tipe data kalau Python sudah otomatis tahu?" Nah, ini bagian yang seringkali tidak disadari dan sangat penting!

Python perlu tahu tipe data karena cara ia mengolah informasi sangat tergantung pada tipenya. Contoh paling sederhana adalah tanda +:

  • Jika Anda menambahkan dua angka (Integer atau Float), Python akan melakukan operasi penjumlahan.
  • Jika Anda "menambahkan" dua teks (String), Python tidak akan menjumlahkannya, melainkan menggabungkannya (menyambungnya).
  • Mencoba menggabungkan tipe data yang tidak cocok bisa menyebabkan kesalahan fatal pada program Anda.

Untuk mengetahui tipe data dari sebuah variabel atau nilai apa pun, Python menyediakan sebuah "fungsi" khusus bernama type(). Anda cukup menulis type() dan meletakkan variabel atau nilai yang ingin Anda periksa di dalam tanda kurungnya. 

# Mari kita lihat tipe data dari variabel-variabel kita
nama_lengkap = "Dian Permata"
umur_siswa = 17
berat_badan = 55.7
apakah_punya_laptop = True

# Menggunakan print() untuk menampilkan tipe data
print("Tipe data dari nama_lengkap adalah:", type(nama_lengkap))
print("Tipe data dari umur_siswa adalah:", type(umur_siswa))
print("Tipe data dari berat_badan adalah:", type(berat_badan))
print("Tipe data dari apakah_punya_laptop adalah:", type(apakah_punya_laptop))

print("\n--- Contoh Operasi Berbeda Tipe ---")
print("Penjumlahan angka:", 5 + 3) # Hasilnya 8
print("Penggabungan teks:", "Selamat" + " " + "Belajar") # Hasilnya "Selamat Belajar"

# print("Ini akan ERROR:", "Angka" + 10) # Ini akan menyebabkan error karena menggabungkan teks dan angka

Outputnya:

Tipe data dari nama_lengkap adalah: 
Tipe data dari umur_siswa adalah: 
Tipe data dari berat_badan adalah: 
Tipe data dari apakah_punya_laptop adalah: 

--- Contoh Operasi Berbeda Tipe ---
Penjumlahan angka: 8
Penggabungan teks: Selamat Belajar

(Catatan: Baris print("Ini akan ERROR:", "Angka" + 10) sengaja dikomentari (#) agar program bisa berjalan tanpa error. Jika Anda menghapus tanda # dan mencobanya, Python akan menampilkan pesan kesalahan karena Anda mencoba "menambahkan" teks dengan angka, yang tidak mungkin dilakukan secara langsung.)

Penjelasan Kode:

Ketika Anda menggunakan type(), Python akan memberitahu Anda jenis "kelas" dari data tersebut. Misalnya, <class 'str'> berarti itu adalah String, <class 'int'> adalah Integer, <class 'float'> adalah Float, dan <class 'bool'> adalah Boolean.

Contoh penjumlahan angka (5 + 3) dan penggabungan teks ("Selamat" + " " + "Belajar") menunjukkan bagaimana Python melakukan operasi yang berbeda tergantung pada tipe datanya. Ini adalah bukti pentingnya Python memahami tipe data dari setiap informasi yang Anda berikan. Jika Anda mencoba mencampur aduk operasi yang tidak cocok (seperti menambahkan string dengan integer), Python akan langsung memberi tahu Anda bahwa ada masalah.

Memahami tipe data adalah langkah awal yang sangat penting agar Anda bisa menulis program yang benar dan menghindari kesalahan yang tidak perlu.
