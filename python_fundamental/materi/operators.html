"Operator" – Alat Aksi di Python

Kita sudah belajar tentang "variabel" sebagai "kotak penyimpanan" dan "tipe data" sebagai "jenis barang" di dalam kotak itu. Sekarang, bayangkan Anda ingin melakukan sesuatu dengan barang-barang di dalam kotak. Mungkin Anda ingin menjumlahkan dua angka, membandingkan dua nama, atau mengubah nilai di dalam kotak. Di sinilah peran Operator.

Operator adalah simbol atau kata kunci khusus yang memberitahu Python untuk melakukan operasi tertentu pada satu atau lebih nilai atau variabel. Mereka seperti "kata kerja" dalam kalimat yang memungkinkan data untuk "beraksi".

Operator Aritmatika: Melakukan Perhitungan Seperti Kalkulator

Operator aritmatika adalah yang paling sering kita temui, karena fungsinya mirip dengan tombol-tombol pada kalkulator Anda. Mereka digunakan untuk melakukan operasi matematika dasar pada angka (baik bilangan bulat atau desimal).

  • + (Penjumlahan): Menambah dua nilai.
  • - (Pengurangan): Mengurangi satu nilai dari nilai lain.
  • * (Perkalian): Mengalikan dua nilai.
  • / (Pembagian): Membagi satu nilai dengan nilai lain. Hasilnya selalu berupa bilangan desimal (float), meskipun hasil pembagiannya bulat.
  • // (Pembagian Bulat / Floor Division): Membagi satu nilai dengan nilai lain, tetapi hanya mengambil bagian bilangan bulatnya saja (membulatkan ke bawah).
  • % (Sisa Bagi / Modulus): Membagi satu nilai dengan nilai lain, dan hasilnya adalah sisa dari pembagian tersebut.
  • ** (Pangkat): Memangkatkan satu nilai dengan nilai lain (misalnya, 2 pangkat 3 ditulis 2 ** 3). 
# Contoh penggunaan operator aritmatika

panjang = 10
lebar = 4
total_belanja = 50000
jumlah_orang = 3

# Penjumlahan dan Pengurangan
jumlah_total = panjang + lebar
sisa_stok = panjang - 3

# Perkalian
luas_persegi_panjang = panjang * lebar

# Pembagian biasa (hasilnya float)
rata_rata_pengeluaran = total_belanja / jumlah_orang # 50000 / 3

# Pembagian bulat (hanya ambil bagian bulatnya)
jumlah_kelompok = 17 // 3 # 17 dibagi 3, hasilnya 5 (bulat)

# Sisa bagi (modulus)
sisa_siswa = 17 % 3 # 17 dibagi 3, sisanya 2

# Pangkat
dua_pangkat_tiga = 2 ** 3 # Sama dengan 2 * 2 * 2

print("Panjang + Lebar:", jumlah_total)
print("Sisa Stok:", sisa_stok)
print("Luas Persegi Panjang:", luas_persegi_panjang)
print("Rata-rata Pengeluaran:", rata_rata_pengeluaran)
print("Jumlah Kelompok (Pembagian Bulat):", jumlah_kelompok)
print("Sisa Siswa (Modulus):", sisa_siswa)
print("Dua Pangkat Tiga:", dua_pangkat_tiga)

# Contoh penggunaan '+' untuk teks (string)
salam = "Selamat"
pesan = "Datang"
gabungan_kata = salam + " " + pesan
print("Gabungan Kata:", gabungan_kata)

Outputnya:

Panjang + Lebar: 14
Sisa Stok: 7
Luas Persegi Panjang: 40
Rata-rata Pengeluaran: 16666.666666666668
Jumlah Kelompok (Pembagian Bulat): 5
Sisa Siswa (Modulus): 2
Dua Pangkat Tiga: 8
Gabungan Kata: Selamat Datang

Penjelasan Kode:

Pada contoh ini, kita melihat bagaimana setiap operator aritmatika bekerja. Perhatikan perbedaan antara / dan //. Ketika total_belanja dibagi jumlah_orang menggunakan tanda /, hasilnya adalah 16666.666... (bilangan desimal), meskipun aslinya kita membagi angka bulat (integer). Namun, ketika 17 dibagi 3 dengan //, hasilnya adalah 5, karena Python hanya mengambil bagian bilangan bulatnya. Operator % sangat berguna untuk mengetahui "sisa" dari suatu pembagian, misalnya untuk memeriksa apakah suatu angka genap atau ganjil (jika dibagi 2 dan sisanya 0, maka angkat tersebut genap). Operator ** digunakan untuk pangkat.

Kita juga melihat bagaimana operator + bisa digunakan untuk menggabungkan dua string (teks). Python memahami bahwa jika Anda menambahkan teks, yang Anda inginkan adalah menyambungkan mereka, bukan menjumlahkan secara matematis. Ini adalah contoh bagaimana Python menyesuaikan perilaku operator tergantung pada tipe data yang digunakannya.

Operator Penugasan: Operator Pengubah Isi Variabel

Kita sudah familiar dengan tanda = yang digunakan untuk "memasukkan" atau "menugaskan" sebuah nilai ke dalam variabel (contoh: usia = 17). Namun, seringkali kita ingin mengubah nilai suatu variabel berdasarkan nilainya sendiri, misalnya menambahkannya dengan angka tertentu. Python menyediakan "operator penugasan gabungan" yang lebih ringkas untuk melakukan ini.

  • = (Penugasan Sederhana): Mengisi variabel dengan nilai baru.
  • += (Tambah dan Tugaskan): Menambahkan nilai ke variabel yang sudah ada, lalu menyimpan hasilnya kembali ke variabel itu. (Contoh: x += 5 sama dengan x = x + 5)
  • -= (Kurang dan Tugaskan): Mengurangi nilai dari variabel, lalu menyimpan hasilnya. (Contoh: x -= 5 sama dengan x = x - 5)
  • *= (Kali dan Tugaskan): Mengalikan variabel dengan nilai, lalu menyimpan hasilnya. (Contoh: x *= 5 sama dengan x = x * 5)
  • /= (Bagi dan Tugaskan): Membagi variabel dengan nilai, lalu menyimpan hasilnya. (Contoh: x /= 5 sama dengan x = x / 5)
  • Dan seterusnya untuk //=, %=, **=. 
# Contoh penggunaan operator penugasan

total_poin = 100
print("Poin awal:", total_poin)

# Menambahkan 20 poin
total_poin += 20 # Sama dengan: total_poin = total_poin + 20
print("Poin setelah tambahan:", total_poin) # Output: 120

# Mengurangi 50 poin karena penalti
total_poin -= 50 # Sama dengan: total_poin = total_poin - 50
print("Poin setelah penalti:", total_poin) # Output: 70

# Menggandakan poin
total_poin *= 2 # Sama dengan: total_poin = total_poin * 2
print("Poin setelah digandakan:", total_poin) # Output: 140

jumlah_siswa = 30
jumlah_siswa /= 2 # Sama dengan: jumlah_siswa = jumlah_siswa / 2
print("Jumlah siswa dibagi dua:", jumlah_siswa) # Output: 15.0 (hasilnya float)

Outputnya:

Poin awal: 100
Poin setelah tambahan: 120
Poin setelah penalti: 70
Poin setelah digandakan: 140
Jumlah siswa dibagi dua: 15.0

Penjelasan Kode:

Operator penugasan gabungan ini adalah cara cepat dan rapi untuk mengubah nilai sebuah variabel. Daripada menulis total_poin = total_poin + 20, kita cukup menulis total_poin += 20. Ini membuat kode lebih singkat, lebih mudah dibaca, dan efisien.

Operator Perbandingan: Membandingkan Nilai dan Mencari Nilai True/False

Operator perbandingan digunakan untuk membandingkan dua nilai. Hasil dari perbandingan ini selalu berupa tipe data Boolean, yaitu True (Benar) atau False (Salah). Ini sangat penting untuk membuat keputusan dalam program Anda, yang akan kita bahas di modul selanjutnya (Kondisional).

  • == (Sama Dengan): Memeriksa apakah terdapat kesamaan antara dua nilai. Perhatikan, ini berbeda dengan = (penugasan)!
  • != (Tidak Sama Dengan): Memeriksa apakah dua nilai tidak bernilai sama.
  • > (Lebih Besar Dari): Memeriksa apakah nilai kiri lebih besar dari nilai kanan.
  • < (Lebih Kecil Dari): Memeriksa apakah nilai kiri lebih kecil dari nilai kanan.
  • >= (Lebih Besar Dari atau Sama Dengan): Memeriksa apakah nilai kiri lebih besar dari atau sama dengan nilai kanan.
  • <= (Lebih Kecil Dari atau Sama Dengan): Memeriksa apakah nilai kiri lebih kecil dari atau sama dengan nilai kanan. 
# Contoh penggunaan operator perbandingan

nilai_siswa = 85
nilai_minimal_lulus = 75
nama_depan_1 = "Andi"
nama_depan_2 = "Budi"

# Membandingkan angka
apakah_lulus = nilai_siswa >= nilai_minimal_lulus
apakah_nilai_sempurna = nilai_siswa == 100
apakah_di_bawah_kkm = nilai_siswa < nilai_minimal_lulus

# Membandingkan teks (huruf besar/kecil berpengaruh!)
apakah_nama_sama = nama_depan_1 == "andi" # Perhatikan huruf kecil 'andi'
apakah_nama_berbeda = nama_depan_1 != nama_depan_2

print("Apakah siswa lulus?", apakah_lulus)
print("Apakah nilai sempurna?", apakah_nilai_sempurna)
print("Apakah nilai di bawah KKM?", apakah_di_bawah_kkm)
print("Apakah nama depan 1 sama dengan 'andi'?", apakah_nama_sama)
print("Apakah nama depan 1 tidak sama dengan nama depan 2?", apakah_nama_berbeda)

Outputnya:

Apakah siswa lulus? True
Apakah nilai sempurna? False
Apakah nilai di bawah KKM? False
Apakah nama depan 1 sama dengan 'andi'? False
Apakah nama depan 1 tidak sama dengan nama depan 2? True

Penjelasan Kode:

Setiap hasil dari operasi perbandingan adalah True atau False. Misalnya, nilai_siswa >= nilai_minimal_lulus menghasilkan True karena 85 memang lebih besar dari atau sama dengan 75. Perhatikan juga bahwa saat membandingkan teks, Python sangat peka terhadap huruf besar dan kecil. "Andi" tidak sama dengan "andi", sehingga apakah_nama_sama menjadi False. Operator perbandingan adalah fondasi untuk membuat program yang dapat "berpikir" dan merespons kondisi yang berbeda.

Operator Logika: Menggabungkan Nilai Boolean

Operator logika digunakan untuk menggabungkan atau membalik nilai-nilai Boolean (True atau False). Mereka memungkinkan kita untuk membuat kondisi yang lebih kompleks.

  • and (Dan): Hasilnya True jika kedua kondisi yang dibandingkan True. Jika salah satu atau keduanya False, maka hasilnya False.
  • or (Atau): Hasilnya True jika salah satu (atau keduanya) kondisi yang dibandingkan True. Hasilnya hanya False jika kedua kondisi False.
  • not (Bukan): Membalik nilai Boolean. Jika kondisinya True, not akan mengubahnya menjadi False, dan sebaliknya. 
# Contoh penggunaan operator logika

punya_kartu_pelajar = True
bawa_buku = False
umur = 17
syarat_umur_masuk = 16

# Menggunakan 'and'
boleh_ikut_ekskul = punya_kartu_pelajar and (umur >= syarat_umur_masuk) # True dan True = True
print("Boleh ikut ekskul?", boleh_ikut_ekskul)

# Menggunakan 'or'
bisa_masuk_perpustakaan = punya_kartu_pelajar or bawa_buku # True atau False = True
print("Bisa masuk perpustakaan?", bisa_masuk_perpustakaan)

# Menggunakan 'not'
tidak_bawa_buku = not bawa_buku # not False = True
print("Tidak bawa buku?", tidak_bawa_buku)

# Contoh kombinasi
apakah_siap_ujian = (punya_kartu_pelajar and bawa_buku) or (umur > 18)
# (True dan False) atau (False) --> False atau False --> False
print("Apakah siap ujian?", apakah_siap_ujian)

Outputnya:

Boleh ikut ekskul? True
Bisa masuk perpustakaan? True
Tidak bawa buku? True
Apakah siap ujian? False

Penjelasan Kode:

Operator logika and, or, dan not sangat berguna untuk membuat keputusan berlapis. and membutuhkan semua kondisi untuk benar agar hasilnya benar. or hanya membutuhkan satu kondisi saja yang benar agar hasilnya benar. Sedangkan not adalah operator "pembalik". Kita juga bisa menggabungkan operator logika dengan tanda kurung () untuk mengatur urutan evaluasinya, mirip seperti perhitungan dalam matematika.

Prioritas Operator: Siapa yang Dikerjakan Lebih Dulu?

Sama seperti dalam pelajaran matematika di mana ada aturan "Kali/Bagi dulu, baru Tambah/Kurang" (urutan operasi), dalam Python juga ada aturan prioritas operator. Python akan mengeksekusi operasi dengan operator yang memiliki prioritas lebih tinggi terlebih dahulu.

Urutan prioritas (dari yang tertinggi ke terendah, beberapa yang umum):

  1. Pangkat (**)
  2. Perkalian (*), Pembagian (/, //, %)
  3. Penjumlahan (+), Pengurangan (-)
  4. Perbandingan (==, !=, >, <, >=, <=)
  5. Logika (not, and, or)

Jika ada operator dengan prioritas yang sama, Python akan mengeksekusinya dari kiri ke kanan. Anda bisa menggunakan tanda kurung () untuk mengubah atau mengesampingkan urutan prioritas standar, memaksa Python untuk mengerjakan bagian di dalam kurung terlebih dahulu. 

# Contoh prioritas operator

hasil_1 = 5 + 2 * 3 # Perkalian dulu, lalu penjumlahan
print("Hasil 1 (5 + 2 * 3):", hasil_1) # 5 + 6 = 11

hasil_2 = (5 + 2) * 3 # Penjumlahan di dalam kurung dulu, lalu perkalian
print("Hasil 2 ((5 + 2) * 3):", hasil_2) # 7 * 3 = 21

hasil_3 = 10 / 2 + 3 # Pembagian dulu, lalu penjumlahan
print("Hasil 3 (10 / 2 + 3):", hasil_3) # 5.0 + 3 = 8.0

kondisi_gabungan = True or False and False # 'and' dulu, lalu 'or'
print("Kondisi Gabungan (True or False and False):", kondisi_gabungan) # True or (False) = True

kondisi_gabungan_2 = (True or False) and False # 'or' di dalam kurung dulu, lalu 'and'
print("Kondisi Gabungan 2 ((True or False) and False):", kondisi_gabungan_2) # (True) and False = False

Outputnya:

Hasil 1 (5 + 2 * 3): 11
Hasil 2 ((5 + 2) * 3): 21
Hasil 3 (10 / 2 + 3): 8.0
Kondisi Gabungan (True or False and False): True
Kondisi Gabungan 2 ((True or False) and False): False

Penjelasan Kode:

Contoh ini dengan jelas menunjukkan bagaimana prioritas operator memengaruhi hasil akhir. Pada hasil_1, perkalian 2 * 3 dikerjakan lebih dulu, baru hasilnya ditambahkan dengan 5. Namun, pada hasil_2, tanda kurung memaksa Python untuk menjumlahkan 5 + 2 terlebih dahulu, baru hasilnya dikalikan dengan 3.

Hal yang sama berlaku untuk operator logika. and memiliki prioritas lebih tinggi daripada or. Oleh karena itu, di kondisi_gabungan, False and False dievaluasi terlebih dahulu dan hasilnya menjadi False, barulah True or False dievaluasi yang menghasilkan nilai True. Ketika kita menambahkan tanda kurung di kondisi_gabungan_2, maka urutannya berubah.

Memahami prioritas operator sangat penting agar perhitungan atau logika di program Anda berjalan sesuai yang Anda inginkan. Jika ragu, selalu gunakan tanda kurung untuk memastikan urutan operasi yang tepat!

Dengan menguasai operator, Anda kini memiliki kemampuan untuk melakukan perhitungan yang kompleks, membandingkan informasi, dan membuat program Anda bertindak lebih dinamis berdasarkan data yang ada!
