Struktur Data (Set) – Kumpulan Item Unik Tanpa Urutan

Kita sudah belajar tentang List (kumpulan item berurutan yang bisa diubah) dan Tuple (kumpulan item berurutan yang tidak bisa diubah). Sekarang, mari kita kenali Set (atau "himpunan" dalam matematika). Set adalah sebuah kumpulan item yang tidak berurutan dan tidak boleh ada item yang kembar (unik).

Bayangkan Anda memiliki sekumpulan kartu remi di tangan, tanpa urutan tertentu, dan Anda tidak akan pernah memiliki dua kartu yang persis sama (misalnya, dua kartu As Hati). Set di Python mirip dengan konsep himpunan dalam matematika: fokusnya adalah pada keberadaan item dan keunikannya, bukan pada urutan atau posisi item tersebut.

Membuat Set: Menggunakan Tanda Kurung Kurawal {}

Untuk membuat sebuah Set di Python, kita menggunakan tanda kurung kurawal ({}) dan meletakkan item-item di dalamnya, dipisahkan dengan tanda koma (,). Jika Anda mencoba memasukkan item yang sama lebih dari satu kali, Set secara otomatis akan menghapus duplikatnya dan hanya menyimpan satu salinan.

Penting: Untuk membuat Set yang kosong, Anda harus menggunakan set(), bukan {}. Karena {} tanpa isi akan dianggap sebagai tipe data lain yang akan kita pelajari nanti (Dictionary). 

# Set berisi angka (dengan duplikat yang akan dihapus)
angka_acak = {1, 5, 2, 5, 8, 2, 10}
print("Set angka acak:", angka_acak) # Perhatikan duplikat 5 dan 2 hilang, urutan bisa berubah

# Set berisi teks (nama buah unik)
buah_unik = {"Apel", "Mangga", "Apel", "Pisang", "Jeruk"}
print("Set buah unik:", buah_unik) # Perhatikan 'Apel' hanya muncul sekali

# Set bisa berisi berbagai jenis tipe data (tapi itemnya harus immutable!)
informasi_unik = {"Budi", 17, 1.70, True, 17} # 17 duplikat
print("Informasi unik:", informasi_unik)

# Cara membuat set kosong
set_kosong = set()
print("Set yang kosong:", set_kosong, ", Tipe:", type(set_kosong))

# Jika pakai {}, itu bukan set kosong!
bukan_set_kosong = {}
print("Ini bukan set kosong:", bukan_set_kosong, ", Tipe:", type(bukan_set_kosong))

# Kita bisa memeriksa tipe data dari set
print("Tipe data dari angka_acak adalah:", type(angka_acak))

Outputnya:

Set angka acak: {1, 2, 5, 8, 10}
Set buah unik: {'Mangga', 'Pisang', 'Apel', 'Jeruk'}
Informasi unik: {True, 'Budi', 17, 1.7}
Set yang kosong: set() , Tipe: <class 'set'>
Ini bukan set kosong: {} , Tipe: <class 'dict'>
Tipe data dari angka_acak adalah: <class 'set'>

Penjelasan Kode:

Pada angka_acak = {1, 5, 2, 5, 8, 2, 10}, meskipun 5 dan 2 ditulis dua kali, Python secara otomatis hanya menyimpan satu salinannya. Hasilnya {1, 2, 5, 8, 10}. Perhatikan juga bahwa urutan item dalam Set yang dicetak bisa berbeda dari urutan Anda memasukkannya, karena Set memang tidak memiliki urutan yang dijamin.

Penting untuk diingat bahwa {} tanpa item di dalamnya adalah untuk dict (Dictionary), bukan set kosong. Untuk membuat set kosong, gunakan set().

Sifat Utama Set: Unik dan Tidak Berurutan

Dua poin ini adalah kunci untuk memahami Set:

  1. Item Unik (Tidak Ada Duplikat): Set secara otomatis memastikan setiap item di dalamnya unik. Ini sangat berguna untuk membersihkan daftar dari item yang berulang.
  2. Tidak Berurutan (Unordered): Item dalam Set tidak memiliki posisi atau indeks tetap. Anda tidak bisa mengakses item dengan nama_set[0] seperti pada List atau Tuple. Ini juga berarti Anda tidak bisa melakukan slicing pada Set.
# Contoh 1: Menghilangkan duplikat dari list
nilai_mentah = [70, 80, 70, 90, 80, 100, 70]
print("List nilai mentah:", nilai_mentah)

# Ubah list menjadi set untuk menghilangkan duplikat
nilai_unik = set(nilai_mentah)
print("Set nilai unik:", nilai_unik)

# Ubah kembali ke list jika urutan diperlukan
nilai_unik_list = list(nilai_unik)
print("List nilai unik (urutan bisa berubah):", nilai_unik_list)

print("\n")

# Contoh 2: Tidak bisa mengakses item berdasarkan indeks
siswa_hadir = {"Andi", "Budi", "Cici"}
# print(siswa_hadir[0]) # Ini akan error! TypeError: 'set' object is not subscriptable

Outputnya:

List nilai mentah: [70, 80, 70, 90, 80, 100, 70]
Set nilai unik: {70, 80, 90, 100}
List nilai unik (urutan bisa berubah): [70, 80, 90, 100]


Traceback (most recent call last):
  File "", line 13, in 
TypeError: 'set' object is not subscriptable

Penjelasan Kode:

Pada Contoh 1, ketika nilai_mentah (sebuah List) diubah menjadi set(nilai_mentah), semua angka yang berulang (70 dan 80) dihapus, hanya menyisakan satu salinan dari setiap nilai unik. Jika Anda perlu list tanpa duplikat, ini adalah trik yang sangat cepat! Setelah itu, Anda bisa mengubahnya kembali ke list jika Anda butuh struktur yang berurutan, meskipun urutannya mungkin tidak sama dengan list aslinya.

Contoh 2 menunjukkan bahwa Anda tidak bisa menggunakan indeks ([0]) pada Set. Python akan protes dengan pesan TypeError: 'set' object is not subscriptable, yang berarti "objek set tidak bisa diakses dengan indeks".

Menambah dan Menghapus Item dari Set: add() dan remove()/discard()

Meskipun Set tidak berurutan, Anda tetap bisa menambah dan menghapus item dari dalamnya (Set bersifat mutable, seperti List).

  • add(item): Menambahkan item baru ke Set. Jika item sudah ada, tidak akan terjadi apa-apa (karena Set tidak menerima duplikat).
  • remove(item): Menghapus item dari Set. Jika item yang ingin dihapus tidak ada dalam Set, akan terjadi error (KeyError).
  • discard(item): Menghapus item dari Set. Mirip dengan remove(), tetapi jika item yang ingin dihapus tidak ada, tidak akan terjadi error. Ini lebih "aman".
  • pop(): Menghapus dan mengembalikan item acak dari Set. Karena Set tidak berurutan, Anda tidak tahu item mana yang akan dihapus.
  • clear(): Menghapus semua item dari Set, membuat Set kosong.
daftar_mapel = {"Matematika", "Fisika"}
print("Daftar mapel awal:", daftar_mapel)

# Menambah item
daftar_mapel.add("Kimia")
print("Setelah add 'Kimia':", daftar_mapel)

daftar_mapel.add("Fisika") # Menambah item yang sudah ada, tidak ada perubahan
print("Setelah add 'Fisika' lagi:", daftar_mapel)

# Menghapus item dengan remove()
daftar_mapel.remove("Fisika")
print("Setelah remove 'Fisika':", daftar_mapel)

# daftar_mapel.remove("Biologi") # Ini akan error! KeyError: 'Biologi'

# Menghapus item dengan discard() (lebih aman)
daftar_mapel.discard("Kimia")
print("Setelah discard 'Kimia':", daftar_mapel)

daftar_mapel.discard("Biologi") # Tidak error jika item tidak ada
print("Setelah discard 'Biologi' (yang tidak ada):", daftar_mapel)

# Menghapus item acak dengan pop()
item_dihapus_pop = daftar_mapel.pop()
print("Setelah pop:", daftar_mapel)
print("Item yang dihapus oleh pop:", item_dihapus_pop)

# Mengosongkan Set
daftar_mapel.clear()
print("Setelah clear():", daftar_mapel)

Outputnya:

Daftar mapel awal: {'Matematika', 'Fisika'}
Setelah add 'Kimia': {'Matematika', 'Kimia', 'Fisika'}
Setelah add 'Fisika' lagi: {'Matematika', 'Kimia', 'Fisika'}
Setelah remove 'Fisika': {'Matematika', 'Kimia'}
Setelah discard 'Kimia': {'Matematika'}
Setelah discard 'Biologi' (yang tidak ada): {'Matematika'}
Setelah pop: set()
Item yang dihapus oleh pop: Matematika
After clear(): set()

Penjelasan Kode:

add() akan menambahkan item baru jika belum ada. remove() dan discard() berfungsi untuk menghapus, dengan discard() menjadi pilihan yang lebih baik jika Anda tidak yakin apakah item tersebut ada di Set atau tidak (untuk menghindari error). pop() berguna untuk mengambil dan menghapus item secara acak dari Set.

Operasi Himpunan Matematika: Kekuatan Sejati Set!

Kekuatan sebenarnya dari Set muncul saat Anda melakukan operasi himpunan (seperti yang mungkin Anda pelajari di pelajaran matematika). Ini sangat efisien untuk membandingkan dua atau lebih kumpulan item.

  • union() (Gabungan): Menggabungkan semua item unik dari kedua Set. Operatornya adalah |.
  • intersection() (Irisan): Mengambil item yang ada di kedua Set. Operatornya adalah &.
  • difference() (Selisih): Mengambil item yang ada di Set pertama tetapi TIDAK ada di Set kedua. Operatornya adalah -.
  • symmetric_difference() (Selisih Simetris): Mengambil item yang unik di setiap Set (ada di Set pertama atau kedua, tetapi TIDAK di keduanya). Operatornya adalah ^.
  • issubset() (Bagian dari): Memeriksa apakah semua item di satu Set ada di Set lain.
  • issuperset() (Mengandung): Memeriksa apakah satu Set mengandung semua item dari Set lain.
siswa_kls_a = {"Andi", "Budi", "Cici", "Dina"}
siswa_kls_b = {"Cici", "Dina", "Eko", "Fina"}

print("Siswa Kelas A:", siswa_kls_a)
print("Siswa Kelas B:", siswa_kls_b)

# Gabungan (Union) - Semua siswa yang ada di A atau B (tanpa duplikat)
semua_siswa = siswa_kls_a.union(siswa_kls_b)
# Atau pakai operator: semua_siswa = siswa_kls_a | siswa_kls_b
print("Semua siswa (gabungan):", semua_siswa)

# Irisan (Intersection) - Siswa yang ada di A DAN B
siswa_bersama = siswa_kls_a.intersection(siswa_kls_b)
# Atau pakai operator: siswa_bersama = siswa_kls_a & siswa_kls_b
print("Siswa yang ada di kedua kelas (irisan):", siswa_bersama)

# Selisih (Difference) - Siswa di A TAPI TIDAK di B
siswa_hanya_di_a = siswa_kls_a.difference(siswa_kls_b)
# Atau pakai operator: siswa_hanya_di_a = siswa_kls_a - siswa_kls_b
print("Siswa hanya di Kelas A:", siswa_hanya_di_a)

# Selisih Simetris (Symmetric Difference) - Siswa yang unik di A ATAU B
siswa_unik_di_tiap_kelas = siswa_kls_a.symmetric_difference(siswa_kls_b)
# Atau pakai operator: siswa_unik_di_tiap_kelas = siswa_kls_a ^ siswa_kls_b
print("Siswa yang unik di masing-masing kelas:", siswa_unik_di_tiap_kelas)

# Subset dan Superset
set_kecil = {"Andi", "Budi"}
apakah_set_kecil_subset = set_kecil.issubset(siswa_kls_a)
print(f"Apakah {set_kecil} adalah subset dari {siswa_kls_a}?", apakah_set_kecil_subset)

apakah_kls_a_superset = siswa_kls_a.issuperset(set_kecil)
print(f"Apakah {siswa_kls_a} adalah superset dari {set_kecil}?", apakah_kls_a_superset)

Outputnya:

Siswa Kelas A: {'Andi', 'Budi', 'Cici', 'Dina'}
Siswa Kelas B: {'Cici', 'Dina', 'Eko', 'Fina'}
Semua siswa (gabungan): {'Andi', 'Dina', 'Cici', 'Fina', 'Eko', 'Budi'}
Siswa yang ada di kedua kelas (irisan): {'Cici', 'Dina'}
Siswa hanya di Kelas A: {'Andi', 'Budi'}
Siswa yang unik di masing-masing kelas: {'Andi', 'Eko', 'Fina', 'Budi'}
Apakah {'Andi', 'Budi'} adalah subset dari {'Andi', 'Budi', 'Cici', 'Dina'}? True
Apakah {'Andi', 'Budi', 'Cici', 'Dina'} adalah superset dari {'Andi', 'Budi'}? True

Penjelasan Kode:

Contoh-contoh ini menunjukkan bagaimana Set sangat efisien untuk melakukan operasi perbandingan himpunan. Ini sangat berguna dalam skenario di mana Anda perlu menemukan elemen umum antara dua daftar, menghapus duplikat, atau mengidentifikasi elemen yang hanya ada di satu kumpulan saja.

Kapan Menggunakan Set?

Anda akan memilih Set ketika:

  1. Anda hanya peduli dengan keberadaan item, bukan urutannya.
  2. Anda ingin memastikan tidak ada item duplikat. Set secara otomatis menangani ini.
  3. Anda sering melakukan operasi himpunan seperti gabungan, irisan, atau selisih. Operasi ini jauh lebih cepat pada Set dibandingkan jika Anda melakukannya secara manual pada List.
  4. Anda perlu memeriksa apakah suatu item ada dalam kumpulan data dengan sangat cepat. Pemeriksaan item in set_saya sangat efisien di Set.

Apa yang Mungkin Tidak Disadari: Elemen Set Harus Tidak Dapat Diubah (Immutable)

Ini adalah batasan penting pada Set: Set hanya bisa berisi item yang tidak dapat diubah (immutable). Artinya, Anda tidak bisa memasukkan List atau Set lain langsung ke dalam sebuah Set. Anda juga tidak bisa memasukkan Dictionary (yang akan kita pelajari selanjutnya).

Mengapa? Karena Set menggunakan sesuatu yang disebut "hashing" untuk menyimpan dan menemukan item dengan cepat, dan hanya objek yang tidak dapat diubah yang memiliki hash yang stabil. 

# Contoh item yang BISA dimasukkan ke Set
set_berisi_immutable = {10, "Halo", 3.14, True, (1, 2, 3)} # Tuple bisa karena immutable
print(set_berisi_immutable)

# Contoh item yang TIDAK BISA dimasukkan ke Set (akan error!)
list_didalam_set = [1, 2]
# set_dengan_list = {1, 2, list_didalam_set} # Ini akan error! TypeError: unhashable type: 'list'
# print(set_dengan_list)

set_didalam_set = {1, 2}
# set_dengan_set_lain = {10, set_didalam_set} # Ini akan error! TypeError: unhashable type: 'set'
# print(set_dengan_set_lain)

Outputnya:

{True, (1, 2, 3), 10, 3.14, 'Halo'}

Penjelasan Kode:

Anda bisa memasukkan angka, string, Boolean, dan bahkan Tuple ke dalam Set karena semuanya adalah tipe data yang tidak dapat diubah (immutable). Namun, jika Anda mencoba memasukkan List atau Set lain, Python akan mengeluarkan TypeError karena mereka adalah tipe data yang dapat diubah (mutable).
