"Input" – Menerima Masukan dari Pengguna

Sebelumnya, kita sudah belajar bagaimana program Python "berbicara" kepada kita melalui Output menggunakan print(). Sekarang, saatnya kita belajar bagaimana kita bisa "berbicara" kembali kepada program. Proses di mana program menerima informasi dari pengguna (misalnya, melalui ketikan di keyboard) inilah yang disebut Input.

Dalam Python, ada satu alat utama yang kita gunakan untuk meminta input dari pengguna, yaitu fungsi input().

Meminta Nama: Penggunaan Dasar input()

Fungsi input() akan "menunggu" sampai pengguna mengetik sesuatu dan menekan tombol Enter. Apapun yang diketik pengguna akan diambil oleh input() dan bisa kita simpan ke dalam sebuah variabel. 

# Program meminta nama Anda
nama = input()

# Lalu program menampilkan nama yang Anda masukkan
print("Halo,", nama + "! Senang bertemu denganmu.")

Outputnya (contoh interaksi):

Budi Santoso  <-- Ini adalah ketikan pengguna
Halo, Budi Santoso! Senang bertemu denganmu.

Penjelasan Kode:

Pada baris nama = input(), program berhenti sejenak dan menunggu Anda mengetik sesuatu. Saat Anda mengetik Budi Santoso dan menekan Enter, teks tersebut akan diambil oleh fungsi input() dan langsung disimpan ke dalam variabel nama. Kemudian, print("Halo,", nama + "! Senang bertemu denganmu.") akan menampilkan pesan sambutan yang berisi nama yang baru saja Anda masukkan.

Memahami Rahasia input()

Ini adalah salah satu hal penting dan sering tidak disadari tentang input(): Apapun yang Anda ketik, Python akan selalu menganggapnya sebagai sebuah teks (tipe data string). Bahkan jika Anda mengetik angka, Python akan melihatnya sebagai teks, bukan sebagai angka yang bisa dihitung.

Mari kita buktikan dengan contoh:

# Meminta umur dan mencoba mengetahui tipenya
print("Berapa umur Anda?")
umur_anda = input()

print("Umur Anda adalah:", umur_anda)
print("Tipe data dari umur yang Anda masukkan adalah:", type(umur_anda))

# Mari kita coba menambahkan angka ke umur_anda (akan terjadi error!)
# hasil_penambahan = umur_anda + 5 # Ini akan error!
# print(hasil_penambahan)

Outputnya (contoh interaksi):

Berapa umur Anda?
17              <-- Saya mengetik angka 17
Umur Anda adalah: 17
Tipe data dari umur yang Anda masukkan adalah: <class 'str'>

Penjelasan Kode:

Meskipun kita mengetik angka 17, type(umur_anda) menunjukkan bahwa Python menyimpannya sebagai <class 'str'> (string). Karena umur_anda adalah teks, operasi penjumlahan umur_anda + 5 tidak bisa dilakukan dan akan menyebabkan error, persis seperti ketika kita mencoba menambahkan "Angka" dengan 10 di modul Tipe Data. Ini adalah poin yang harus selalu diingat saat bekerja dengan input dari pengguna.

Menghadapi Angka dari Pengguna

Karena input() selalu menghasilkan string, jika kita ingin melakukan perhitungan matematika dengan angka yang dimasukkan pengguna, kita harus mengonversi string tersebut menjadi tipe data angka (int atau float) terlebih dahulu. Kita sudah belajar tentang fungsi int() dan float() di modul Konversi Tipe Data. 

# Meminta angka dan mengonversinya untuk perhitungan
print("Masukkan sebuah angka bulat pertama:")
angka_1_text = input()
angka_1 = int(angka_1_text) # Konversi teks ke angka bulat

print("Masukkan sebuah angka bulat kedua:")
angka_2_text = input()
angka_2 = int(angka_2_text) # Konversi teks ke angka bulat

# Sekarang kita bisa menjumlahkannya karena sudah menjadi angka
hasil_penjumlahan = angka_1 + angka_2
print("Hasil penjumlahan kedua angka adalah:", hasil_penjumlahan)

print("\n")

# Contoh dengan angka desimal
print("Masukkan tinggi badan Anda dalam meter (misal 1.75):")
tinggi_text = input()
tinggi_float = float(tinggi_text) # Konversi teks ke angka desimal

berat_badan_kg = 60
# Ini adalah contoh sederhana untuk demonstrasi
# Nanti kita bisa minta input berat badan juga
indeks_massa_tubuh = berat_badan_kg / (tinggi_float ** 2)
print("Indeks Massa Tubuh Anda adalah:", indeks_massa_tubuh)

Outputnya (contoh interaksi):

Masukkan sebuah angka bulat pertama:
15            <-- Pengguna mengetik 15
Masukkan sebuah angka bulat kedua:
20            <-- Pengguna mengetik 20
Hasil penjumlahan kedua angka adalah: 35

Masukkan tinggi badan Anda dalam meter (misal 1.75):
1.68          <-- Pengguna mengetik 1.68
Indeks Massa Tubuh Anda adalah: 21.25877843846618

Penjelasan Kode:

Pada contoh ini, kita pertama kali meminta input angka bulat. Meskipun pengguna mengetik 15, input() tetap menyimpannya sebagai "15" (string). Barulah pada baris angka_1 = int(angka_1_text), kita menggunakan int() untuk mengubah "15" menjadi angka 15 yang sebenarnya, siap untuk perhitungan. Hal yang sama berlaku untuk angka_2.

Untuk tinggi badan, karena kita mengharapkan angka desimal, kita menggunakan float(tinggi_text) untuk mengubah input string "1.68" menjadi angka desimal 1.68. Setelah dikonversi, kita bisa melakukan perhitungan matematika yang melibatkan pangkat dan pembagian dengan benar.

Memberi Petunjuk Jelas: Menggunakan input() dengan Pesan Prompt

Agar pengguna tidak bingung apa yang harus mereka ketik, fungsi input() memiliki kemampuan untuk menampilkan sebuah pesan sebelum menunggu input. Pesan ini disebut "prompt". Anda cukup menuliskannya di dalam tanda kurung (). 

# input() tanpa prompt (kurang jelas bagi pengguna)
# nama_hewan = input()
# print("Nama hewan favorit Anda:", nama_hewan)

print("\n--- Contoh dengan Prompt Jelas ---")

# input() dengan prompt
nama_siswa = input("Siapa nama lengkap Anda? ")
print("Halo, ", nama_siswa)

umur_siswa = input("Berapa usia Anda saat ini? ")
# Jangan lupa konversi jika ingin angka!
umur_sekarang = int(umur_siswa)
tahun_depan = umur_sekarang + 1
print("Tahun depan Anda akan berusia", tahun_depan, "tahun.")

Outputnya (contoh interaksi):

--- Contoh dengan Prompt Jelas ---
Siapa nama lengkap Anda? Siti Rahmawati  <-- Pengguna mengetik
Halo,  Siti Rahmawati
Berapa usia Anda saat ini? 16            <-- Pengguna mengetik
Tahun depan Anda akan berusia 17 tahun.

Penjelasan Kode:

Perhatikan bagaimana input("Siapa nama lengkap Anda? ") langsung menampilkan pertanyaan di layar dan kursor menunggu di sebelahnya. Ini jauh lebih ramah pengguna dibandingkan hanya kursor yang berkedip-kedip tanpa petunjuk. Pengguna langsung tahu apa yang harus diisikan.

Setelah pengguna mengetik 16 dan menekan Enter, umur_siswa akan berisi "16". Kita kemudian mengubahnya menjadi int(umur_siswa) agar bisa dijumlahkan dengan 1 untuk mendapatkan usia tahun depan.

Keindahan input: Program yang Berdialog

Kemampuan untuk menerima input dari pengguna mengubah program dari sekadar menjalankan perintah satu arah menjadi sebuah "dialog". Ini membuka banyak sekali kemungkinan:

  • Membuat program kuis interaktif.
  • Membuat kalkulator yang meminta angka dari pengguna.
  • Membuat game sederhana di mana pengguna memasukkan pilihan.
  • Mengumpulkan data dari pengguna untuk disimpan atau diproses lebih lanjut.

Saat Anda mulai menggabungkan input() dengan variabel, tipe data, operator, dan konversi tipe data, Anda sudah bisa membuat program sederhana yang cukup canggih dan berguna!

Dengan modul ini, Anda sekarang bisa membuat program yang tidak hanya berbicara, tetapi juga mendengarkan! Ini adalah langkah besar dalam perjalanan pemrograman Anda.
