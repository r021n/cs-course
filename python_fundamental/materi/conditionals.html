"Kondisi" – Membuat Keputusan di Python

Dalam kehidupan sehari-hari, kita sering membuat keputusan berdasarkan kondisi tertentu:

  • "Jika hari ini cerah, maka saya akan pergi bermain di luar."
  • "Jika saya lapar, maka saya akan makan. Jika tidak, maka saya akan belajar."
  • "Jika nilai saya A, maka saya akan senang sekali. Jika nilainya B, maka saya cukup senang. Jika tidak (nilai saya C, D, atau E), maka saya harus belajar lebih giat."

Dalam pemrograman, kita juga perlu membuat program melakukan hal yang sama. Kita perlu memberitahu program untuk menjalankan kode tertentu hanya jika suatu kondisi terpenuhi. Inilah inti dari Kondisional atau Pernyataan Kondisional. Python menggunakan kata kunci khusus seperti if, elif, dan else untuk membuat keputusan ini.

if: Pernyataan "Jika..."

Pernyataan if adalah dasar dari semua keputusan. Ia bekerja seperti ini: "Jika suatu kondisi ini benar, maka jalankan kode di bawah ini." Jika kondisi itu salah, Python akan melewati kode tersebut.

Kondisi yang kita gunakan di sini adalah hasil dari operasi perbandingan (seperti ==, >, <, dll.) atau operasi logika (and, or, not) yang sudah kita pelajari. Ingat, hasil dari perbandingan ini selalu True (benar) atau False (salah).

Perhatikan baik-baik, setelah kondisi if, selalu ada tanda titik dua (:) dan baris kode di bawahnya harus menjorok ke dalam (menggunakan spasi atau Tab). Ini namanya indentasi dan sangat penting di Python! 

# Contoh 1: Cek apakah nilai siswa lulus KKM
nilai_ujian = 80
kkm = 75

if nilai_ujian >= kkm: # Kondisi: apakah 80 lebih besar atau sama dengan 75? (True)
    print("Selamat! Anda dinyatakan LULUS.") # Baris ini akan dijalankan

print("Program selesai.")

print("\n")

# Contoh 2: Jika kondisi tidak terpenuhi
suhu_luar = 20 # Suhu dalam Celsius
if suhu_luar > 25: # Kondisi: apakah 20 lebih besar dari 25? (False)
    print("Hari ini sangat panas!") # Baris ini TIDAK akan dijalankan

print("Cek suhu selesai.")

Outputnya:

Selamat! Anda dinyatakan LULUS.
Program selesai.


Cek suhu selesai.

Penjelasan Kode:

Pada Contoh 1, nilai_ujian >= kkm (yaitu 80 >= 75) hasilnya adalah True. Karena kondisinya benar, baris print("Selamat! Anda dinyatakan LULUS.") yang berada di bawah if dan menjorok ke dalam akan dieksekusi.

Pada Contoh 2, suhu_luar > 25 (yaitu 20 > 25) hasilnya adalah False. Karena kondisinya salah, baris print("Hari ini sangat panas!") yang menjorok ke dalam di bawah if akan diabaikan oleh Python. Python langsung melanjutkan ke baris setelah blok if, yaitu print("Cek suhu selesai.").

if-else: Memberi Pilihan "Jika... Maka... Jika Tidak..."

Pernyataan if-else memberi program dua jalur berbeda. Jika kondisi di if benar, maka satu blok kode akan dijalankan. Jika tidak (kondisi di if salah), maka blok kode yang ada di bawah else akan dijalankan.

Sama seperti if, else juga diikuti oleh tanda titik dua (:) dan kode di bawahnya harus menjorok ke dalam. 

# Meminta input dari pengguna
usia_pengunjung_str = input("Berapa usia Anda? ")
usia_pengunjung = int(usia_pengunjung_str) # Konversi ke integer

batas_usia_masuk = 17

if usia_pengunjung >= batas_usia_masuk: # Kondisi: apakah usia >= 17?
    print("Anda diizinkan masuk.")
    print("Selamat menikmati acara!")
else: # Jika kondisi di atas salah (usia < 17)
    print("Maaf, usia Anda belum mencukupi.")
    print("Anda tidak diizinkan masuk.")

print("Terima kasih sudah berkunjung.")

Outputnya (Contoh 1: Usia 18):

Berapa usia Anda? 18
Anda diizinkan masuk.
Selamat menikmati acara!
Terima kasih sudah berkunjung.

Outputnya (Contoh 2: Usia 15):

Berapa usia Anda? 15
Maaf, usia Anda belum mencukupi.
Anda tidak diizinkan masuk.
Terima kasih sudah berkunjung.

Penjelasan Kode:

Program akan meminta usia pengguna. Jika usia yang dimasukkan 18, maka kondisi usia_pengunjung >= batas_usia_masuk akan menjadi True, dan blok kode di bawah if akan dieksekusi.

Namun, jika usia yang dimasukkan 15, kondisi tersebut menjadi False. Oleh karena itu, Python akan melewati blok if dan langsung melompat ke blok else, menjalankan pesan "Maaf, usia Anda belum mencukupi."

if-elif-else: Banyak Pilihan "Jika... Atau Jika... Atau Jika Tidak..."

Bagaimana jika ada lebih dari dua kemungkinan? Misalnya, Anda ingin memberikan peringkat (grade) A, B, C, D, atau E berdasarkan nilai siswa. Di sinilah elif (kependekan dari "else if") sangat berguna. Anda bisa menambahkan berapa pun jumlah elif yang Anda butuhkan untuk memeriksa kondisi tambahan secara berurutan.

Python akan memeriksa kondisi if terlebih dahulu. Jika salah, ia akan memeriksa elif pertama. Jika itu juga salah, ia akan memeriksa elif berikutnya, dan seterusnya. Jika tidak ada satu pun if atau elif yang benar, maka blok else (jika ada) akan dijalankan sebagai pilihan terakhir.

Penting: Urutan elif sangat memengaruhi hasil! Kondisi yang lebih spesifik atau lebih ketat biasanya diletakkan di atas. 

# Program penentuan nilai huruf (grade)
nilai_rata_rata_str = input("Masukkan nilai rata-rata Anda: ")
nilai_rata_rata = float(nilai_rata_rata_str) # Konversi ke float, karena bisa desimal

if nilai_rata_rata >= 90:
    print("Nilai Anda: A (Sangat Baik)")
elif nilai_rata_rata >= 80: # Hanya akan diperiksa jika nilai < 90
    print("Nilai Anda: B (Baik)")
elif nilai_rata_rata >= 70: # Hanya akan diperiksa jika nilai < 80
    print("Nilai Anda: C (Cukup)")
elif nilai_rata_rata >= 60: # Hanya akan diperiksa jika nilai < 70
    print("Nilai Anda: D (Kurang)")
else: # Jika semua kondisi di atas salah (nilai < 60)
    print("Nilai Anda: E (Sangat Kurang), perlu remedial.")

print("Pengecekan nilai selesai.")

Outputnya (Contoh 1: Nilai 92):

Masukkan nilai rata-rata Anda: 92
Nilai Anda: A (Sangat Baik)
Pengecekan nilai selesai.

Outputnya (Contoh 2: Nilai 78):

Masukkan nilai rata-rata Anda: 78
Nilai Anda: C (Cukup)
Pengecekan nilai selesai.

Penjelasan Kode:

Jika Anda memasukkan 92, kondisi nilai_rata_rata >= 90 akan True, dan Python akan mencetak "Nilai Anda: A". Python kemudian akan melewati semua elif dan else lainnya dan langsung ke bagian print("Pengecekan nilai selesai.").

Jika Anda memasukkan 78:

  1. nilai_rata_rata >= 90 (78 >= 90) -> False
  2. elif nilai_rata_rata >= 80 (78 >= 80) -> False
  3. elif nilai_rata_rata >= 70 (78 >= 70) -> True! Python akan mencetak "Nilai Anda: C" dan langsung melewati sisa elif dan else.

Inilah mengapa urutan sangat penting. Jika elif nilai_rata_rata >= 70 diletakkan di atas elif nilai_rata_rata >= 80, maka nilai 85 akan menghasilkan C padahal seharusnya B!

Kondisi yang Lebih Canggih dengan Operator Logika

Kita bisa membuat kondisi yang sangat kompleks dengan menggabungkan operator perbandingan dan operator logika (and, or, not).

  • and: Kedua sisi harus Benar.
  • or: Salah satu sisi (atau keduanya) harus Benar.
  • not: Membalik kebenaran. 
# Program cek kelayakan pinjaman
penghasilan_bulanan_str = input("Masukkan penghasilan bulanan Anda: ")
penghasilan_bulanan = int(penghasilan_bulanan_str)

usia_str = input("Masukkan usia Anda: ")
usia = int(usia_str)

status_menikah_str = input("Apakah Anda sudah menikah? (ya/tidak): ").lower() # Ubah ke huruf kecil

# Syarat kelayakan:
# 1. Penghasilan di atas 5 juta DAN usia di atas 20
# ATAU
# 2. Sudah menikah DAN penghasilan di atas 3 juta

if (penghasilan_bulanan > 5000000 and usia > 20) or \
   (status_menikah_str == "ya" and penghasilan_bulanan > 3000000):
    print("Selamat! Anda memenuhi syarat untuk mengajukan pinjaman.")
else:
    print("Maaf, Anda belum memenuhi syarat pinjaman.")

print("Terima kasih.")

Outputnya (Contoh 1: Penghasilan 6 juta, Usia 22, Belum menikah): 

Masukkan penghasilan bulanan Anda: 6000000
Masukkan usia Anda: 22
Apakah Anda sudah menikah? (ya/tidak): tidak
Selamat! Anda memenuhi syarat untuk mengajukan pinjaman.
Terima kasih.

Outputnya (Contoh 2: Penghasilan 4 juta, Usia 19, Sudah menikah): 

Masukkan penghasilan bulanan Anda: 4000000
Masukkan usia Anda: 19
Apakah Anda sudah menikah? (ya/tidak): ya
Selamat! Anda memenuhi syarat untuk mengajukan pinjaman.
Terima kasih.

Outputnya (Contoh 3: Penghasilan 2 juta, Usia 25, Belum menikah): 

Masukkan penghasilan bulanan Anda: 2000000
Masukkan usia Anda: 25
Apakah Anda sudah menikah? (ya/tidak): tidak
Maaf, Anda belum memenuhi syarat pinjaman.
Terima kasih.

Penjelasan Kode:

Kondisi di sini sangat kompleks. Python akan mengevaluasi bagian dalam kurung terlebih dahulu karena prioritas operator.

  • penghasilan_bulanan > 5000000 and usia > 20 akan bernilai True jika kedua kondisi di dalamnya benar.
  • status_menikah_str == "ya" and penghasilan_bulanan > 3000000 akan bernilai True jika kedua kondisi di dalamnya benar.

Kemudian, hasil dari kedua bagian tersebut akan digabungkan dengan or. Jika salah satu dari dua kelompok kondisi besar itu benar, maka seluruh kondisi if akan menjadi True. Ini menunjukkan betapa kuatnya operator logika untuk membuat keputusan yang sangat spesifik dan berlapis.

Pentingnya Indentasi

Di modul ini sudah berkali-kali menyebutkan indentasi. Ini bukan sekadar masalah kerapian, tapi adalah aturan sintaksis yang WAJIB dipatuhi di Python. Python menggunakan indentasi untuk mengetahui blok kode mana yang "milik" suatu if, elif, atau else. Biasanya, indentasi menggunakan 4 spasi.

Jika indentasi Anda salah, Python akan langsung mengeluarkan error yang disebut IndentationError, dan program Anda tidak akan bisa berjalan. 

# Contoh Indentasi yang BENAR
usia = 18
if usia >= 17:
    print("Anda dewasa.") # Ini di dalam blok 'if'
    print("Bisa memiliki KTP.") # Ini juga di dalam blok 'if'
print("Pengecekan usia selesai.") # Ini di luar blok 'if'

print("\n")

# Contoh Indentasi yang SALAH (akan menyebabkan error!)
# usia_lain = 16
# if usia_lain >= 17:
# print("Maaf, Anda belum dewasa.") # ERROR: Indentasi salah, tidak menjorok!
#    print("Tetap semangat!") # ERROR: Indentasi salah, tidak konsisten dengan baris di atasnya

# print("Program berakhir.")

Outputnya (dari kode yang benar):

Anda dewasa.
Bisa memiliki KTP.
Pengecekan usia selesai.

(Catatan: Bagian kode yang salah sengaja dikomentari (#) agar program bisa berjalan tanpa error. Jika Anda menghapus tanda pagar # pada baris yang salah dan menjalankannya, Anda akan melihat pesan IndentationError.)

Penjelasan Kode:

Pada contoh yang benar, print("Anda dewasa.") dan print("Bisa memiliki KTP.") sama-sama menjorok ke dalam dengan jumlah spasi yang sama (misalnya 4 spasi). Ini memberitahu Python bahwa kedua baris ini adalah bagian dari blok kode if. Sedangkan print("Pengecekan usia selesai.") tidak menjorok, sehingga Python tahu itu adalah kode yang akan dijalankan setelah if selesai, tidak peduli apakah kondisi if benar atau salah.

Kesalahan indentasi adalah salah satu error yang paling umum ditemui pemula di Python. Selalu pastikan Anda konsisten dengan jumlah spasi untuk indentasi!

Apa yang Mungkin Tidak Disadari: Kondisi "Truthy" dan "Falsy"

Ini adalah konsep yang lebih lanjut tapi sangat berguna: di Python, tidak hanya True dan False yang bisa digunakan dalam kondisi if. Python memiliki konsep "Truthy" (dianggap benar) dan "Falsy" (dianggap salah). Ini berarti Anda bisa menulis kondisi tanpa perlu operator perbandingan eksplisit.

Secara umum:

  • Nilai "Falsy" (dianggap False):
    • Angka nol (0 dan 0.0)
    • String kosong ("" atau '')
    • Nilai khusus None (berarti "tidak ada nilai")
  • Semua nilai lainnya dianggap "Truthy" (dianggap True). 
nama_pengguna = input("Masukkan nama Anda (kosongkan jika tidak ada): ")
jumlah_produk = 0 # Angka 0 akan dianggap False
pesan_khusus = "Selamat datang kembali!" # String tidak kosong akan dianggap True

print("\n--- Contoh Truthy/Falsy ---")

if nama_pengguna: # Jika nama_pengguna tidak kosong (Truthy)
    print(f"Halo, {nama_pengguna}!")
else:
    print("Anda belum memasukkan nama.")

if jumlah_produk: # Jika jumlah_produk tidak nol (Truthy)
    print(f"Ada {jumlah_produk} produk di keranjang.")
else:
    print("Keranjang belanja Anda kosong.")

if pesan_khusus: # Jika pesan_khusus tidak kosong (Truthy)
    print(pesan_khusus)
else:
    print("Tidak ada pesan khusus.")

Outputnya (Contoh 1: Nama diisi, jumlah produk 0): 

Masukkan nama Anda (kosongkan jika tidak ada): Budi

--- Contoh Truthy/Falsy ---
Halo, Budi!
Keranjang belanja Anda kosong.
Selamat datang kembali!

Outputnya (Contoh 2: Nama kosong, jumlah produk 5): 

Masukkan nama Anda (kosongkan jika tidak ada): 

--- Contoh Truthy/Falsy ---
Anda belum memasukkan nama.
Keranjang belanja Anda kosong. <-- ini karena saya set jumlah_produk = 0
Selamat datang kembali!

Penjelasan Kode:

Pada if nama_pengguna:, Python secara otomatis mengecek apakah string nama_pengguna itu kosong atau tidak. Jika tidak kosong (pengguna mengetik sesuatu), maka nama_pengguna dianggap True dan blok if dijalankan. Jika kosong, dianggap False dan blok else dijalankan.

Sama halnya dengan if jumlah_produk:. Karena jumlah_produk adalah 0, Python menganggapnya False, sehingga blok else yang dieksekusi. Jika jumlah_produk adalah 5, maka akan dianggap True.

Konsep truthy/falsy ini memungkinkan Anda menulis kode yang lebih ringkas dan elegan untuk memeriksa keberadaan data atau jumlah nol.

Kondisional Bersarang (Nested Conditionals): Keputusan di Dalam Keputusan

Anda bisa meletakkan pernyataan kondisional ( if, elif, else) di dalam blok kode kondisional lainnya. Ini disebut kondisional bersarang. Ini berguna ketika Anda perlu membuat keputusan berlapis. 

# Program pemeriksaan kelayakan masuk tim olahraga
nilai_rapor_str = input("Masukkan nilai rata-rata rapor Anda: ")
nilai_rapor = float(nilai_rapor_str)

tinggi_badan_str = input("Masukkan tinggi badan Anda (cm): ")
tinggi_badan = int(tinggi_badan_str)

jenis_kelamin = input("Jenis kelamin (L/P): ").upper() # Ubah ke huruf besar

print("\n--- Hasil Pengecekan ---")

if nilai_rapor >= 75:
    print("Nilai rapor memenuhi syarat.")
    if jenis_kelamin == "L":
        if tinggi_badan >= 170:
            print("Anda memenuhi syarat untuk tim basket putra!")
        else:
            print("Untuk tim basket putra, tinggi badan belum mencukupi.")
    elif jenis_kelamin == "P":
        if tinggi_badan >= 160:
            print("Anda memenuhi syarat untuk tim basket putri!")
        else:
            print("Untuk tim basket putri, tinggi badan belum mencukupi.")
    else:
        print("Jenis kelamin tidak valid.")
else:
    print("Maaf, nilai rapor Anda belum memenuhi syarat.")

print("Pengecekan selesai.")

Outputnya (Contoh: Rapor 80, Tinggi 175, L):

Masukkan nilai rata-rata rapor Anda: 80
Masukkan tinggi badan Anda (cm): 175
Jenis kelamin (L/P): L

--- Hasil Pengecekan ---
Nilai rapor memenuhi syarat.
Anda memenuhi syarat untuk tim basket putra!
Pengecekan selesai.

Penjelasan Kode:

Pertama, program memeriksa if nilai_rapor >= 75. Jika kondisi ini benar, barulah program masuk ke blok kode di dalamnya (yang menjorok dua kali). Di dalam blok itu, ada if jenis_kelamin == "L". Jika itu benar, program masuk lagi ke blok di dalamnya (menjorok tiga kali) untuk memeriksa tinggi badan.

Perhatikan bagaimana setiap lapisan kondisional memiliki tingkat indentasi yang berbeda. Ini adalah cara Python memahami struktur keputusan yang berlapis. Kondisional bersarang memungkinkan Anda untuk membuat program yang sangat cerdas dalam merespons berbagai skenario.
