"Perulangan" – Membuat Program Melakukan Tugas Berulang

Bayangkan Anda diminta untuk menulis angka 1 sampai 100, atau menghitung jumlah siswa di 10 kelas yang berbeda. Jika Anda harus menulis perintah print() seratus kali, atau menulis perhitungan yang sama berulang kali, itu akan sangat melelahkan dan rawan kesalahan.

Di sinilah Perulangan (atau Loop) menjadi penyelamat! Perulangan adalah instruksi yang memberitahu Python untuk mengulang satu atau beberapa baris kode berkali-kali. Ada dua jenis perulangan utama di Python: for dan while, masing-masing dengan kegunaannya sendiri.

Perulangan for: Mengulang Berdasarkan Jumlah atau Kumpulan

Perulangan for digunakan ketika Anda tahu berapa kali Anda ingin mengulang sesuatu, atau ketika Anda ingin mengulang untuk setiap "barang" dalam sebuah kumpulan. Ia seperti Anda berkata, "Untuk setiap barang di kotak ini, lakukan A." atau "Untuk setiap angka dari 1 sampai 5, lakukan B."

Python menggunakan fungsi khusus bernama range() untuk membantu kita membuat urutan angka yang dapat diulang oleh perulangan for.

Mengulang Dengan Angka: Menggunakan range()

Fungsi range() sangat berguna untuk membuat perulangan yang berjalan sejumlah tertentu.

  • range(jumlah): Akan menghasilkan angka dari 0 sampai jumlah-1.
  • range(mulai, akhir): Akan menghasilkan angka dari mulai sampai akhir-1.
  • range(mulai, akhir, langkah): Akan menghasilkan angka dari mulai sampai akhir-1, dengan melompati sejumlah langkah. 
# Contoh 1: Menghitung dari 0 sampai 4
print("--- Menghitung 0 sampai 4 ---")
for angka in range(5): # range(5) menghasilkan 0, 1, 2, 3, 4
    print(angka)

print("\n") # Baris kosong untuk keterbacaan

# Contoh 2: Menghitung dari 1 sampai 5
print("--- Menghitung 1 sampai 5 ---")
for i in range(1, 6): # range(1, 6) menghasilkan 1, 2, 3, 4, 5
    print(i)

print("\n")

# Contoh 3: Menghitung mundur dari 5 sampai 1
print("--- Menghitung Mundur dari 5 ---")
for hitung_mundur in range(5, 0, -1): # range(5, 0, -1) menghasilkan 5, 4, 3, 2, 1
    print(hitung_mundur)

print("\n")

# Contoh 4: Menampilkan pesan berulang kali
print("--- Pesan Berulang ---")
for _ in range(3): # Jika kita tidak peduli dengan angkanya, pakai underscore (_)
    print("Semangat belajar Python!")

Penjelasan Kode:

Pada Contoh 1, for angka in range(5): memberitahu Python untuk mengulang kode di dalamnya sebanyak 5 kali. Setiap kali perulangan, variabel angka akan diisi dengan angka berikutnya dari range(5) (yaitu 0, lalu 1, lalu 2, dst.).

Pada Contoh 2, range(1, 6) membuat urutan angka dimulai dari 1 hingga sebelum 6, yaitu 1, 2, 3, 4, 5.

Pada Contoh 3, range(5, 0, -1) adalah trik untuk menghitung mundur. Dimulai dari 5, berakhir sebelum 0, dengan langkah -1 (berarti mengurangi 1 setiap kali).

Pada Contoh 4, kita menggunakan _ (underscore) sebagai nama variabel di for _ in range(3):. Ini adalah kebiasaan umum di Python ketika kita hanya ingin mengulang sejumlah kali dan tidak benar-benar peduli dengan nilai angka di setiap perulangan. Ini memberitahu pembaca bahwa variabel itu tidak akan digunakan di dalam perulangan.

Mengulang Untuk Setiap Karakter dalam Teks (String)

String (teks) juga merupakan sebuah "kumpulan" karakter. Perulangan for bisa digunakan untuk "mengunjungi" setiap karakter satu per satu. 

# Contoh 1: Mencetak setiap huruf dari nama
nama_siswa = "Python"
print("--- Huruf demi Huruf ---")
for huruf in nama_siswa:
    print(huruf)

print("\n")

# Contoh 2: Menghitung jumlah huruf vokal dalam sebuah kata
kata = "Komputer"
jumlah_vokal = 0
for karakter in kata:
    # Mengubah karakter menjadi huruf kecil agar mudah dibandingkan
    if karakter.lower() == 'a' or karakter.lower() == 'i' or \
       karakter.lower() == 'u' or karakter.lower() == 'e' or \
       karakter.lower() == 'o':
        jumlah_vokal += 1 # Tambahkan 1 ke jumlah_vokal

print(f"Kata: '{kata}' memiliki {jumlah_vokal} huruf vokal.")

Outputnya:

--- Huruf demi Huruf ---
P
y
t
h
o
n

Kata: 'Komputer' memiliki 3 huruf vokal.

Penjelasan Kode:

Pada Contoh 1, for huruf in nama_siswa: akan membuat perulangan berjalan sebanyak jumlah huruf di dalam nama_siswa. Pada setiap putaran, variabel huruf akan diisi dengan karakter berikutnya dari string Python (P, lalu y, lalu t, dst.) hingga semua karakter selesai.

Pada Contoh 2, kita mengulang setiap karakter dalam kata. Di dalam perulangan, kita menggunakan if (kondisional) untuk memeriksa apakah karakter tersebut adalah huruf vokal (setelah diubah ke huruf kecil menggunakan metode .lower()). Jika ya, kita menggunakan operator penugasan += untuk menambahkan 1 ke jumlah_vokal. Ini menunjukkan bagaimana perulangan seringkali digabungkan dengan kondisional untuk melakukan tugas yang lebih cerdas.

Perulangan while: Mengulang Selama Kondisi Benar

Perulangan while sedikit berbeda. Ia akan terus mengulang selama suatu kondisi yang Anda berikan tetap True (benar). Ini seperti Anda berkata, "Selama ada makanan di kulkas, saya akan terus makan."

Kunci dari perulangan while adalah memastikan bahwa kondisi yang Anda berikan suatu saat akan menjadi False, agar perulangan bisa berhenti. Jika tidak, Anda akan membuat "perulangan tak terbatas" (infinite loop) yang tidak akan pernah berhenti! 

# Contoh 1: Menghitung dari 1 sampai 5 menggunakan while
angka_saat_ini = 1
print("--- Menghitung 1 sampai 5 (While) ---")
while angka_saat_ini <= 5: # Kondisi: apakah angka_saat_ini kurang dari atau sama dengan 5?
    print(angka_saat_ini)
    angka_saat_ini += 1 # Penting! Ini mengubah nilai agar kondisi suatu saat False

print("\n")

# Contoh 2: Meminta input sampai pengguna memasukkan jawaban yang benar
jawaban_benar = "Python"
tebakan_pengguna = "" # Variabel awal agar loop bisa dimulai

print("--- Tebak Nama Bahasa Pemrograman ---")
while tebakan_pengguna != jawaban_benar: # Selama tebakan belum benar, ulangi
    tebakan_pengguna = input("Tebak nama bahasa pemrograman favorit saya: ")
    if tebakan_pengguna != jawaban_benar:
        print("Tebakan salah, coba lagi!")
    else:
        print("Selamat! Tebakan Anda benar!")

print("Permainan tebak-tebakan selesai.")

Outputnya (Contoh 1):

--- Menghitung 1 sampai 5 (While) ---
1
2
3
4

Outputnya (Contoh 2: Contoh interaksi):

--- Tebak Nama Bahasa Pemrograman ---
Tebak nama bahasa pemrograman favorit saya: Java
Tebakan salah, coba lagi!
Tebak nama bahasa pemrograman favorit saya: C++
Tebakan salah, coba lagi!
Tebak nama bahasa pemrograman favorit saya: Python
Selamat! Tebakan Anda benar!
Permainan tebak-tebakan selesai.

Penjelasan Kode:

Pada Contoh 1, while angka_saat_ini <= 5: akan terus berjalan selama angka_saat_ini kurang dari atau sama dengan 5. Yang paling penting adalah baris angka_saat_ini += 1. Ini adalah "penjaga" yang memastikan bahwa pada suatu titik, angka_saat_ini akan melebihi 5 (menjadi 6), membuat kondisi False, dan perulangan berhenti. Jika baris ini tidak ada, angka_saat_ini akan selalu 1, dan program akan terus mencetak 1 selamanya (perulangan tak terbatas!).

Pada Contoh 2, perulangan while tebakan_pengguna != jawaban_benar: akan terus meminta input dari pengguna selama tebakan yang dimasukkan belum sama dengan "Python". Begitu pengguna mengetik "Python", kondisi tebakan_pengguna != jawaban_benar akan menjadi False, dan perulangan akan berhenti. Ini sangat berguna untuk memvalidasi input atau membuat menu interaktif.

Menghentikan Perulangan Dini: Perintah break

Terkadang, Anda ingin perulangan berhenti sepenuhnya bahkan sebelum kondisinya menjadi False atau sebelum semua item dalam kumpulan selesai diulang. Perintah break digunakan untuk hal ini. Ketika Python bertemu dengan break, ia akan langsung keluar dari perulangan terdekat dan melanjutkan eksekusi kode setelah perulangan tersebut. 

# Contoh 1: Mencari angka dan berhenti jika sudah ketemu
target_angka = 7
print("--- Mencari Angka 7 ---")
for angka in range(1, 11): # Akan mencoba angka 1 sampai 10
    print(f"Sedang memeriksa angka: {angka}")
    if angka == target_angka:
        print(f"Angka {target_angka} ditemukan! Berhenti mencari.")
        break # Langsung keluar dari loop
print("Pencarian selesai.")

print("\n")

# Contoh 2: Loop tak terbatas yang dihentikan oleh input 'exit'
print("--- Chat Sederhana (ketik 'exit' untuk keluar) ---")
while True: # Ini adalah perulangan tak terbatas
    pesan_anda = input("Anda: ")
    if pesan_anda.lower() == "exit": # Ubah input ke huruf kecil untuk perbandingan
        print("Bot: Sampai jumpa!")
        break # Keluar dari perulangan
    else:
        print(f"Bot: Anda bilang '{pesan_anda}'.")
print("Sesi chat berakhir.")

Outputnya (Contoh 1):

Sedang memeriksa angka: 7

Anda: Apa kabar?

Penjelasan Kode:

Pada Contoh 1, perulangan for akan terus berjalan. Begitu angka mencapai 7, kondisi if angka == target_angka menjadi True. Program mencetak pesan ditemukan, lalu bertemu break. Seketika itu juga, perulangan for berhenti, dan program langsung lompat ke print("Pencarian selesai."), tanpa lagi memeriksa angka 8, 9, dan 10.

Pada Contoh 2, while True: adalah cara membuat perulangan yang secara default akan berjalan selamanya. Namun, di dalamnya ada kondisi if pesan_anda.lower() == "exit": yang jika terpenuhi, akan menjalankan break dan menghentikan perulangan. Ini sering digunakan dalam program yang terus berjalan sampai pengguna memberi perintah khusus untuk keluar.

Melewatkan Iterasi Saat Ini: Perintah continue

Berbeda dengan break yang menghentikan seluruh perulangan, perintah continue akan melewatkan sisa kode di dalam perulangan untuk putaran (iterasi) saat ini, dan langsung melompat ke putaran berikutnya. 

# Contoh 1: Hanya mencetak angka ganjil
print("--- Hanya Angka Ganjil dari 1 sampai 10 ---")
for angka in range(1, 11):
    if angka % 2 == 0: # Jika angka genap (sisa bagi 2 adalah 0)
        print(f"Melewatkan angka genap: {angka}")
        continue # Lewati sisa kode di bawah, langsung ke iterasi berikutnya
    print(f"Angka ganjil: {angka}") # Baris ini hanya dijalankan jika angka ganjil

Outputnya:

--- Hanya Angka Ganjil dari 1 sampai 10 ---
Angka ganjil: 1
Melewatkan angka genap: 2
Angka ganjil: 3
Melewatkan angka genap: 4
Angka ganjil: 5
Melewatkan angka genap: 6
Angka ganjil: 7
Melewatkan angka genap: 8
Angka ganjil: 9
Melewatkan angka genap: 10

Penjelasan Kode:

Pada setiap putaran, if angka % 2 == 0 akan memeriksa apakah angka adalah genap. Jika True, ia mencetak pesan Melewatkan angka genap lalu bertemu continue. Ketika continue dieksekusi, Python tidak akan menjalankan print(f"Angka ganjil: {angka}") untuk angka genap tersebut, melainkan langsung melompat ke putaran berikutnya dari perulangan for. Jika angka ganjil, kondisi if menjadi False, continue tidak dijalankan, dan print(f"Angka ganjil: {angka}") tetap dieksekusi.

else dengan Perulangan: Sesuatu yang Tidak Biasa (Tapi Sangat Berguna!)

Ini adalah salah satu fitur Python yang mungkin tidak Anda temukan di banyak bahasa pemrograman lain, dan seringkali tidak disadari: perulangan for dan while juga bisa memiliki blok else!

Blok else yang berpasangan dengan perulangan akan dieksekusi hanya jika perulangan selesai dengan cara yang "normal" (yaitu, tidak dihentikan oleh perintah break). 

# Contoh 1: Mencari nama dalam daftar (misalnya nanti List)
# Untuk saat ini, bayangkan saja kita sedang mencari huruf dalam sebuah kata
kata_rahasia = "kucing"
cari_huruf = input("Cari huruf apa dalam kata 'kucing'? ")

ditemukan = False # Variabel untuk menandai apakah huruf ditemukan

for huruf in kata_rahasia:
    if huruf == cari_huruf:
        print(f"Huruf '{cari_huruf}' ditemukan di kata!")
        ditemukan = True
        break # Hentikan perulangan karena sudah ketemu
# Blok else ini akan dijalankan HANYA jika loop selesai tanpa 'break'
else:
    print(f"Maaf, huruf '{cari_huruf}' tidak ditemukan di kata.")

print("Pencarian selesai.")

Outputnya (Contoh 1: Cari 'c'):

Cari huruf apa dalam kata 'kucing'? c
Huruf 'c' ditemukan di kata!
Pencarian selesai.

Outputnya (Contoh 2: Cari 'z'):

Cari huruf apa dalam kata 'kucing'? z
Maaf, huruf 'z' tidak ditemukan di kata.
Pencarian selesai.

Penjelasan Kode:

Ketika Anda mencari huruf 'c':

  1. Perulangan dimulai.
  2. Ketika huruf menjadi 'c', kondisi if huruf == cari_huruf menjadi True.
  3. Pesan ditemukan dicetak, ditemukan diubah menjadi True, dan break dieksekusi.
  4. Karena break menghentikan perulangan, blok else tidak dijalankan.

Ketika Anda mencari huruf 'z':

  1. Perulangan berjalan untuk semua huruf 'k', 'u', 'c', 'i', 'n', 'g'.
  2. Kondisi if huruf == cari_huruf tidak pernah menjadi True.
  3. break tidak pernah dieksekusi.
  4. Perulangan selesai secara "normal" (semua item sudah diperiksa).
  5. Karena perulangan selesai normal, blok else pun dijalankan, mencetak pesan tidak ditemukan.

Fitur else dengan perulangan ini sangat rapi untuk skenario "cari sesuatu, jika ketemu lakukan A dan berhenti, jika tidak ketemu sama sekali lakukan B setelah selesai mencari".

Perulangan Bersarang (Nested Loops): Loop di Dalam Loop

Sama seperti kondisional, Anda juga bisa meletakkan satu perulangan di dalam perulangan lainnya. Ini disebut perulangan bersarang (nested loops). Ini sangat berguna untuk tugas-tugas yang melibatkan "grid" atau "tabel", di mana Anda perlu mengulang sesuatu untuk setiap baris dan juga untuk setiap kolom. 

# Contoh: Membuat tabel perkalian sederhana
print("--- Tabel Perkalian Sederhana (3x3) ---")

for i in range(1, 4): # Ini adalah perulangan LUAR (untuk baris, i akan 1, 2, 3)
    for j in range(1, 4): # Ini adalah perulangan DALAM (untuk kolom, j akan 1, 2, 3)
        # print(f"{i} x {j} = {i*j}") # Akan mencetak setiap hasil di baris baru
        print(f"{i}x{j}={i*j}\t", end="") # \t adalah "tab", untuk jarak antar kolom
    print() # Cetak baris baru setelah perulangan dalam selesai (pindah ke baris berikutnya)

print("\n")

# Contoh: Membuat pola bintang persegi
sisi_persegi = 5
print("--- Pola Persegi Bintang ---")
for baris in range(sisi_persegi): # Untuk setiap baris
    for kolom in range(sisi_persegi): # Untuk setiap kolom di baris itu
        print("* ", end="") # Cetak bintang dan spasi, tanpa pindah baris
    print() # Pindah ke baris baru setelah satu baris bintang selesai

Outputnya:

--- Tabel Perkalian Sederhana (3x3) ---
1x1=1	1x2=2	1x3=3	
2x1=2	2x2=4	2x3=6	
3x1=3	3x2=6	3x3=9	

--- Pola Persegi Bintang ---
* * * * * 
* * * * * 
* * * * * 
* * * * * 
* * * * *

Penjelasan Kode:

Pada contoh tabel perkalian:

  • Perulangan luar for i in range(1, 4) akan berjalan 3 kali.
  • Setiap kali perulangan luar berjalan, perulangan dalam for j in range(1, 4) akan berjalan penuh 3 kali.
  • Jadi, total ada 3 x 3 = 9 kali operasi print().
  • print(f"{i}x{j}={i*j}\t", end="") mencetak hasil perkalian diikuti dengan karakter tab (\t) untuk membuat jarak, dan end="" penting agar tidak langsung pindah baris.
  • print() setelah perulangan dalam selesai berfungsi untuk pindah ke baris baru, sehingga hasil berikutnya akan tercetak di baris yang berbeda, membentuk tabel.

Perulangan bersarang adalah alat yang sangat ampuh untuk memproses data dalam bentuk tabel, membuat pola, atau melakukan tugas-tugas yang memerlukan iterasi ganda.
