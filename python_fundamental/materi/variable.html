"Variabel" – Kotak Penyimpanan Data di Program Python

Bayangkan Anda memiliki sebuah kotak kosong di rumah, dan Anda memberi label pada kotak tersebut, misalnya "Buku Pelajaran". Di dalam kotak itu, Anda bisa menyimpan buku apa saja, dan Anda bisa mengganti isinya kapan pun Anda mau. Saat Anda ingin mengambil atau melihat buku di dalamnya, Anda hanya perlu mencari kotak dengan label "Buku Pelajaran".

Dalam pemrograman Python, Variabel itu persis seperti "kotak penyimpanan" berlabel tadi. Variabel adalah nama yang kita berikan untuk sebuah tempat di memori komputer yang bisa kita gunakan untuk menyimpan berbagai jenis informasi, seperti angka, teks, atau data lainnya. Informasi yang disimpan dalam variabel bisa berubah-ubah selama program berjalan.

Menyimpan dan Menampilkan Informasi

Untuk membuat sebuah variabel, kita hanya perlu memberi nama pada "kotak" itu, lalu menggunakan tanda sama dengan (=) untuk "memasukkan" informasi ke dalamnya. Setelah itu, kita bisa menggunakan nama variabel tersebut di mana pun kita membutuhkan informasi yang disimpannya, termasuk untuk ditampilkan menggunakan perintah print(). 

# Ini adalah variabel yang menyimpan nama seseorang
nama_siswa = "Budi Santoso"

# Ini adalah variabel yang menyimpan umur
umur_siswa = 16

# Kita bisa mencetak informasi dari variabel
print("Nama:", nama_siswa)
print("Umur:", umur_siswa, "tahun")

# Komentar ini menjelaskan kegunaan variabel 'nilai_mtk'
nilai_mtk = 90
print("Nilai Matematika:", nilai_mtk)

Outputnya:

Nama: Budi Santoso
Umur: 16 tahun
Nilai Matematika: 90

Penjelasan Kode:

Pada baris nama_siswa = "Budi Santoso", kita membuat sebuah variabel bernama nama_siswa dan "mengisi"nya dengan teks "Budi Santoso". Demikian pula, umur_siswa = 16 membuat variabel umur_siswa yang diisi dengan angka 16.

Ketika kita menggunakan print("Nama:", nama_siswa), Python akan melihat nama_siswa, lalu mengambil isi dari "kotak" itu (yaitu "Budi Santoso"), dan menampilkannya bersamaan dengan teks "Nama:". Sama halnya dengan umur_siswa dan nilai_mtk. Kita bisa memanggil nama variabel berkali-kali tanpa perlu menuliskan ulang datanya.

Aturan Penamaan Variabel (Agar Python Tidak Bingung!)

Sama seperti memberi label pada kotak, ada beberapa aturan dan kebiasaan baik saat memberi nama variabel agar Python mengerti dan program kita mudah dibaca:

  • Hanya Huruf, Angka, dan Garis Bawah (_): Nama variabel hanya boleh terdiri dari huruf (a-z, A-Z), angka (0-9), dan karakter garis bawah (_).
  • Tidak Boleh Diawali Angka: Nama variabel tidak boleh dimulai dengan angka. 1nama itu salah, tapi nama1 itu benar.
  • Peka Huruf Besar/Kecil: Python sangat peka terhadap huruf besar dan kecil. nama_siswa itu berbeda dengan Nama_Siswa atau NAMA_SISWA. Mereka dianggap sebagai tiga variabel yang berbeda!
  • Hindari Kata Khusus Python: Ada beberapa kata yang sudah dipakai oleh Python untuk perintah-perintah khusus (seperti print, if, for). Kita tidak boleh menggunakan kata-kata ini sebagai nama variabel. Jika Anda mencobanya, Python akan menanggapinya dengan pesan kesalahan/error.
  • Gunakan Nama yang Jelas: Selalu coba berikan nama variabel yang menggambarkan isinya. Daripada x = 10, lebih baik jumlah_buku = 10. Ini membuat kode Anda lebih mudah dipahami oleh Anda sendiri di masa depan, atau oleh orang lain yang membaca kode Anda. Kebiasaan umum di Python adalah menggunakan _ (garis bawah) untuk memisahkan kata, seperti nama_lengkap atau nilai_rata_rata. 
# Contoh nama variabel yang BENAR:
nama_depan = "Ani"
total_harga = 25000
_jenis_produk = "Pakaian" # Boleh dimulai dengan underscore
nilai_UAS = 88

# Contoh nama variabel yang SALAH (akan menimbulkan error):
# 1tahun = 2024 # ERROR: Tidak boleh dimulai dengan angka
# for = "loop"   # ERROR: 'for' adalah kata kunci Python
# Nama Depan = "Budi" # ERROR: Tidak boleh ada spasi

print("Nama Depan:", nama_depan)
print("Total Harga:", total_harga)

Outputnya:

Nama Depan: Ani
Total Harga: 25000

Penjelasan Kode:

Contoh di atas menunjukkan nama variabel yang sesuai dengan aturan Python. Ketika Anda mencoba membuat variabel seperti 1tahun atau Nama Depan, Python akan mengeluarkan pesan kesalahan karena melanggar aturan penamaan. Python butuh nama yang jelas dan tidak ambigu untuk "kotaknya".

Variabel Bisa Berubah Isi (Bahkan Tipe Informasi!)

Salah satu hal menarik tentang variabel di Python adalah Anda bisa mengganti isi "kotak" kapan saja. Bahkan, Anda bisa menyimpan jenis informasi yang berbeda di kotak yang sama! 

# Awalnya, variabel 'pesan' berisi teks salam
pesan = "Selamat pagi!"
print(pesan)

# Kemudian, kita ganti isi variabel 'pesan' dengan teks lain
pesan = "Halo semua, ayo belajar Python."
print(pesan)

# Sekarang, kita ubah 'pesan' untuk menyimpan angka
pesan = 12345
print(pesan)

# Bahkan bisa diubah lagi ke teks
pesan = "Angka di atas adalah kode rahasia."
print(pesan)

Outputnya:

Selamat pagi!
Halo semua, ayo belajar Python.
12345
Angka di atas adalah kode rahasia.

Penjelasan Kode:

Pada awalnya, variabel pesan berisi teks "Selamat pagi!". Ketika kita mencetaknya, itulah yang muncul. Namun, pada baris berikutnya, kita "memasukkan" teks yang berbeda ke dalam variabel pesan. Isi yang lama akan tergantikan. Python tidak keberatan jika Anda mengubah isi variabel, bahkan jika Anda menggantinya dari teks menjadi angka (seperti pada pesan = 12345) atau sebaliknya. Python secara otomatis akan menyesuaikan diri dengan jenis data terbaru yang Anda masukkan. Ini membuat Python sangat fleksibel!

Menyalin Isi Variabel (Bukan Kotaknya!)

Ketika Anda membuat satu variabel sama dengan variabel lain, Anda sebenarnya sedang menyalin isi dari variabel pertama ke variabel kedua, bukan membuat kedua variabel itu saling terhubung. Bayangkan Anda punya dua kotak: Kotak A dan Kotak B. Anda mengambil isi dari Kotak A, lalu menyalinnya ke Kotak B. Setelah itu, jika Anda mengubah isi Kotak A, isi Kotak B tidak akan ikut berubah karena mereka adalah kotak yang terpisah. 

nilai_ulangan_ani = 85
nilai_ulangan_budi = nilai_ulangan_ani # Budi menyalin nilai Ani

print("Nilai Ani sebelum diubah:", nilai_ulangan_ani)
print("Nilai Budi (setelah menyalin Ani):", nilai_ulangan_budi)

# Ani memperbaiki ulangannya, nilainya berubah
nilai_ulangan_ani = 95

print("Nilai Ani setelah diubah:", nilai_ulangan_ani)
print("Nilai Budi (tetap sama):", nilai_ulangan_budi) # Nilai Budi tidak berubah

Outputnya:

Nilai Ani sebelum diubah: 85
Nilai Budi (setelah menyalin Ani): 85
Nilai Ani setelah diubah: 95
Nilai Budi (tetap sama): 85

Penjelasan Kode:

Pada awalnya, nilai_ulangan_ani adalah 85. Ketika kita menulis nilai_ulangan_budi = nilai_ulangan_ani, Python mengambil nilai 85 dari nilai_ulangan_ani dan menyimpannya ke dalam nilai_ulangan_budi. Jadi, sekarang nilai_ulangan_budi juga berisi 85.

Kemudian, ketika nilai_ulangan_ani diubah menjadi 95, itu hanya memengaruhi isi "kotak" nilai_ulangan_ani. "Kotak" nilai_ulangan_budi tetap berisi nilai yang disalin sebelumnya, yaitu 85, karena tidak ada lagi hubungan langsung di antara keduanya setelah proses penyalinan nilai. Ini adalah konsep penting untuk dipahami agar tidak kebingungan saat program Anda menjadi lebih kompleks.

Dengan memahami konsep variabel ini, Anda sudah memiliki pemahaman dasar yang kuat untuk menyimpan dan mengelola informasi dalam program Python Anda!
