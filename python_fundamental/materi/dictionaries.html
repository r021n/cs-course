Struktur Data (Dictionary) – Menyimpan Informasi dengan "Kunci" dan "Nilai"

Kita sudah mengenal List (kumpulan item berurutan) dan Tuple (kumpulan item berurutan yang tidak bisa diubah). Sekarang, bayangkan sebuah kamus (dictionary) di mana Anda mencari arti sebuah kata. Anda tidak mencari berdasarkan nomor halaman, tetapi berdasarkan "kata" itu sendiri. Atau, bayangkan buku telepon, di mana Anda mencari nomor telepon seseorang berdasarkan "nama" orang tersebut.

Dalam Python, Dictionary (sering disingkat dict) adalah struktur data yang bekerja mirip dengan kamus atau buku telepon. Dictionary menyimpan data dalam bentuk pasangan "kunci" dan "nilai" (key-value pairs). Setiap "kunci" itu unik dan berfungsi sebagai "nama" atau "identitas" untuk mengakses "nilai" yang bersesuaian. Anda tidak perlu mengingat posisi (indeks) seperti pada List, cukup ingat kuncinya.

Dictionary bersifat mutable (dapat diubah), artinya Anda bisa menambah, menghapus, atau mengubah pasangan kunci-nilai di dalamnya setelah dibuat.

Membuat Dictionary: Menggunakan Kurung Kurawal {} dengan Pasangan Kunci:Nilai

Untuk membuat sebuah Dictionary, kita menggunakan tanda kurung kurawal ({}) dan meletakkan pasangan kunci: nilai di dalamnya, dipisahkan dengan tanda koma (,). Kunci dan nilai dipisahkan oleh tanda titik dua (:). 

# Dictionary data diri siswa
data_siswa = {
    "nama": "Budi Santoso",
    "umur": 17,
    "kelas": "XII IPA 1",
    "sudah_lulus": False
}
print("Data siswa:", data_siswa)

# Dictionary daftar harga barang
daftar_harga = {
    "pensil": 2000,
    "pulpen": 3500,
    "buku": 8000
}
print("Daftar harga:", daftar_harga)

# Dictionary bisa punya nilai dengan tipe data apapun, bahkan List atau Dictionary lain!
biodata_lengkap = {
    "nama": "Siti Rahmawati",
    "usia": 16,
    "hobi": ["Membaca", "Melukis", "Menulis"],
    "nilai_rapor": {
        "Matematika": 90,
        "Fisika": 85
    }
}
print("Biodata lengkap:", biodata_lengkap)

# Membuat dictionary kosong
dict_kosong = {}
print("Dictionary yang kosong:", dict_kosong, ", Tipe:", type(dict_kosong))

# Memeriksa tipe data dari dictionary
print("Tipe data dari data_siswa adalah:", type(data_siswa))

Outputnya:

Data siswa: {'nama': 'Budi Santoso', 'umur': 17, 'kelas': 'XII IPA 1', 'sudah_lulus': False}
Daftar harga: {'pensil': 2000, 'pulpen': 3500, 'buku': 8000}
Biodata lengkap: {'nama': 'Siti Rahmawati', 'usia': 16, 'hobi': ['Membaca', 'Melukis', 'Menulis'], 'nilai_rapor': {'Matematika': 90, 'Fisika': 85}}
Dictionary yang kosong: {} , Tipe: <class 'dict'>
Tipe data dari data_siswa adalah: <class 'dict'>

Penjelasan Kode:

Setiap pasangan kunci: nilai di dalam kurung kurawal {} membentuk sebuah item dalam Dictionary. Misalnya, "nama": "Budi Santoso" berarti kunci adalah "nama" dan nilainya adalah "Budi Santoso". Kunci biasanya berupa teks (string), tapi bisa juga angka atau Tuple. Nilai bisa berupa tipe data apa saja, bahkan List atau Dictionary lain seperti yang terlihat pada biodata_lengkap. Saat mencetak Dictionary, Python menampilkannya dalam format key: value.

Mengakses Nilai dalam Dictionary: Menggunakan Kunci!

Untuk mendapatkan nilai dari sebuah Dictionary, Anda tidak menggunakan indeks angka seperti pada List. Sebaliknya, Anda menggunakan kuncinya di dalam tanda kurung siku ([]). Ini seperti mencari kata di kamus. 

data_pelanggan = {
    "id": "PL001",
    "nama_lengkap": "Ani Nurmala",
    "alamat": "Jl. Merdeka No. 10",
    "telepon": "08123456789"
}

# Mengakses nilai menggunakan kunci
nama_ani = data_pelanggan["nama_lengkap"]
telepon_ani = data_pelanggan["telepon"]

print("Nama Pelanggan:", nama_ani)
print("Nomor Telepon:", telepon_ani)

# Mengakses nilai dari dictionary di dalam dictionary
nilai_mtk_siti = biodata_lengkap["nilai_rapor"]["Matematika"]
print(f"Nilai Matematika Siti: {nilai_mtk_siti}")

# Apa yang terjadi jika kunci tidak ada? ERROR!
# email_ani = data_pelanggan["email"] # Ini akan error! KeyError: 'email'
# print(email_ani)

Outputnya:

Nama Pelanggan: Ani Nurmala
Nomor Telepon: 08123456789
Nilai Matematika Siti: 90

Penjelasan Kode:

Pada nama_ani = data_pelanggan["nama_lengkap"], Python mencari kunci "nama_lengkap" di dalam data_pelanggan dan mengembalikan nilai yang bersesuaian, yaitu "Ani Nurmala". Anda bisa melihat betapa mudahnya mengakses nilai secara langsung dengan namanya.

Sama seperti List, Anda harus berhati-hati saat mengakses kunci yang tidak ada. Jika Anda mencoba mengakses kunci yang tidak ada, Python akan mengeluarkan KeyError, memberitahu Anda bahwa kunci tersebut tidak ditemukan.

Cara Aman Mengakses Nilai: Metode get()

Untuk menghindari KeyError ketika Anda tidak yakin apakah sebuah kunci ada, Anda bisa menggunakan metode .get() pada Dictionary. Jika kunci ditemukan, .get() akan mengembalikan nilainya. Jika tidak ditemukan, ia akan mengembalikan None (yang berarti "tidak ada nilai"), atau nilai default yang bisa Anda tentukan sendiri. 

profil_siswa = {
    "nama": "Rio Fernando",
    "kelas": "XI IPS 2"
}

# Menggunakan get() tanpa nilai default
hobi_rio = profil_siswa.get("hobi") # Kunci 'hobi' tidak ada
print("Hobi Rio:", hobi_rio) # Output: None

# Menggunakan get() dengan nilai default
jurusan_rio = profil_siswa.get("jurusan", "Belum ada informasi jurusan")
print("Jurusan Rio:", jurusan_rio)

nama_rio = profil_siswa.get("nama", "Nama tidak ditemukan")
print("Nama Rio:", nama_rio) # Kunci 'nama' ada, jadi nilainya dikembalikan

Outputnya:

Hobi Rio: None
Jurusan Rio: Belum ada informasi jurusan
Nama Rio: Rio Fernando

Penjelasan Kode:

Ketika profil_siswa.get("hobi") dipanggil, karena kunci "hobi" tidak ada, get() mengembalikan None. Ini jauh lebih baik daripada program berhenti karena error. Anda bisa menggunakan hasil None ini dalam kondisi if untuk menangani kasus ketika informasi tidak ada.

Pada profil_siswa.get("jurusan", "Belum ada informasi jurusan"), jika kunci "jurusan" tidak ada, ia akan mengembalikan string "Belum ada informasi jurusan" sebagai nilai default. Ini sangat berguna untuk memberikan pesan yang lebih ramah kepada pengguna.

Menambah dan Mengubah Pasangan Kunci-Nilai

Dictionary bersifat mutable, yang berarti Anda bisa mengubahnya setelah dibuat.

  • Mengubah nilai: Jika Anda memberikan nilai baru ke kunci yang sudah ada, nilai lama akan diganti.
  • Menambah pasangan baru: Jika Anda memberikan nilai ke kunci yang belum ada, pasangan kunci-nilai baru akan ditambahkan. 
daftar_nilai_mapel = {
    "Matematika": 85,
    "Bahasa Indonesia": 90
}
print("Nilai awal:", daftar_nilai_mapel)

# Mengubah nilai mata pelajaran yang sudah ada
daftar_nilai_mapel["Matematika"] = 92 # Nilai Matematika diubah dari 85 menjadi 92
print("Setelah ubah nilai Matematika:", daftar_nilai_mapel)

# Menambah mata pelajaran baru
daftar_nilai_mapel["Fisika"] = 88 # Kunci "Fisika" belum ada, jadi ditambahkan
print("Setelah tambah nilai Fisika:", daftar_nilai_mapel)

daftar_nilai_mapel["Kimia"] = 75
print("Setelah tambah nilai Kimia:", daftar_nilai_mapel)

Outputnya:

Nilai awal: {'Matematika': 85, 'Bahasa Indonesia': 90}
Setelah ubah nilai Matematika: {'Matematika': 92, 'Bahasa Indonesia': 90}
Setelah tambah nilai Fisika: {'Matematika': 92, 'Bahasa Indonesia': 90, 'Fisika': 88}
Setelah tambah nilai Kimia: {'Matematika': 92, 'Bahasa Indonesia': 90, 'Fisika': 88, 'Kimia': 75}

Penjelasan Kode:

Perhatikan bagaimana baris daftar_nilai_mapel["Matematika"] = 92 langsung mengubah nilai untuk kunci "Matematika". Sementara itu, daftar_nilai_mapel["Fisika"] = 88 menambahkan pasangan baru karena kunci "Fisika" belum ada sebelumnya. Ini menunjukkan fleksibilitas Dictionary dalam mengelola data.

Menghapus Pasangan Kunci-Nilai: del dan pop()

Ada beberapa cara untuk menghapus item (pasangan kunci-nilai) dari Dictionary:

  • del dictionary[kunci]: Menghapus pasangan kunci-nilai berdasarkan kunci yang Anda tentukan. Jika kunci tidak ada, akan terjadi KeyError.
  • pop(kunci): Menghapus pasangan kunci-nilai berdasarkan kunci yang Anda tentukan, dan yang menarik, pop() juga mengembalikan nilai yang dihapus, sehingga Anda bisa menyimpannya ke variabel lain jika diperlukan. Jika kunci tidak ada, akan terjadi KeyError kecuali Anda memberikan nilai default.
  • clear(): Menghapus semua pasangan kunci-nilai dari Dictionary, membuat Dictionary menjadi kosong. 
data_kontak = {
    "Andi": "08123",
    "Budi": "08567",
    "Cici": "08901"
}
print("Kontak awal:", data_kontak)

# Menghapus kontak dengan del
del data_kontak["Budi"]
print("Setelah hapus 'Budi' dengan del:", data_kontak)

# Menghapus kontak dengan pop() dan mengambil nilainya
nomor_cici = data_kontak.pop("Cici")
print("Setelah hapus 'Cici' dengan pop:", data_kontak)
print("Nomor Cici yang dihapus:", nomor_cici)

# Menghapus kunci yang mungkin tidak ada dengan pop() dan default value
nomor_dina = data_kontak.pop("Dina", "Tidak ada di kontak") # Kunci 'Dina' tidak ada
print("Nomor Dina:", nomor_dina) # Output: Tidak ada di kontak
print("Kontak setelah pop 'Dina':", data_kontak)

# Mengosongkan dictionary
data_kontak.clear()
print("Setelah clear():", data_kontak)

Outputnya:

Kontak awal: {'Andi': '08123', 'Budi': '08567', 'Cici': '08901'}
Setelah hapus 'Budi' dengan del: {'Andi': '08123', 'Cici': '08901'}
Setelah hapus 'Cici' dengan pop: {'Andi': '08123'}
Nomor Cici yang dihapus: 08901
Nomor Dina: Tidak ada di kontak
Kontak setelah pop 'Dina': {'Andi': '08123'}
Setelah clear(): {}

Penjelasan Kode:

del data_kontak["Budi"] langsung menghapus pasangan "Budi": "08567". pop("Cici") tidak hanya menghapus, tetapi juga mengembalikan nilai "08901", yang bisa kita simpan ke variabel nomor_cici. Menggunakan pop() dengan nilai default (misalnya, "Tidak ada di kontak") adalah cara aman untuk mencoba menghapus kunci yang mungkin tidak ada, tanpa menyebabkan error.

Melihat Semua Kunci, Nilai, atau Keduanya

Dictionary menyediakan metode yang berguna untuk mendapatkan "pandangan" dari kunci, nilai, atau kedua-duanya:

  • keys(): Mengembalikan pandangan (view) dari semua kunci yang ada di Dictionary.
  • values(): Mengembalikan pandangan (view) dari semua nilai yang ada di Dictionary.
  • items(): Mengembalikan pandangan (view) dari semua pasangan kunci-nilai, di mana setiap pasangan direpresentasikan sebagai Tuple. 
daftar_inventaris = {
    "laptop": 5,
    "mouse": 10,
    "keyboard": 7,
    "monitor": 3
}

# Mendapatkan semua kunci
semua_kunci = daftar_inventaris.keys()
print("Daftar Kunci (Barang):", semua_kunci)

# Mendapatkan semua nilai
semua_nilai = daftar_inventaris.values()
print("Daftar Nilai (Stok):", semua_nilai)

# Mendapatkan semua pasangan kunci-nilai
semua_item = daftar_inventaris.items()
print("Daftar Item (Kunci-Nilai):", semua_item)

# Anda bisa mengubah view ini menjadi List jika perlu
list_kunci = list(semua_kunci)
list_nilai = list(semua_nilai)
list_item = list(semua_item)

print("List Kunci:", list_kunci)
print("List Nilai:", list_nilai)
print("List Item:", list_item)

Outputnya:

Daftar Kunci (Barang): dict_keys(['laptop', 'mouse', 'keyboard', 'monitor'])
Daftar Nilai (Stok): dict_values([5, 10, 7, 3])
Daftar Item (Kunci-Nilai): dict_items([('laptop', 5), ('mouse', 10), ('keyboard', 7), ('monitor', 3)])
List Kunci: ['laptop', 'mouse', 'keyboard', 'monitor']
List Nilai: [5, 10, 7, 3]
List Item: [('laptop', 5), ('mouse', 10), ('keyboard', 7), ('monitor', 3)]

Penjelasan Kode:

keys(), values(), dan items() mengembalikan objek "view". Ini berarti mereka tidak membuat salinan List baru, tetapi mereka "melihat" langsung ke Dictionary. Jika Dictionary berubah, view ini juga akan ikut berubah secara otomatis. Anda bisa mengubah view ini menjadi List dengan list(...) jika Anda ingin salinan yang terpisah atau ingin menggunakannya dengan fungsi List lainnya.

Memeriksa Keberadaan Kunci: Operator in

Anda bisa menggunakan operator in untuk memeriksa apakah sebuah kunci ada di dalam Dictionary. Hasilnya adalah Boolean (True atau False). 

menu_restoran = {
    "Nasi Goreng": 25000,
    "Mie Ayam": 20000,
    "Es Teh": 8000
}

apakah_ada_nasi_goreng = "Nasi Goreng" in menu_restoran
print("Apakah ada 'Nasi Goreng' di menu?", apakah_ada_nasi_goreng)

apakah_ada_pizza = "Pizza" in menu_restoran
print("Apakah ada 'Pizza' di menu?", apakah_ada_pizza)

apakah_tidak_ada_juice = "Juice" not in menu_restoran
print("Apakah tidak ada 'Juice' di menu?", apakah_tidak_ada_juice)

Outputnya:

Apakah ada 'Nasi Goreng' di menu? True
Apakah ada 'Pizza' di menu? False
Apakah tidak ada 'Juice' di menu? True

Penjelasan Kode:

Operator in di Dictionary secara default memeriksa keberadaan kunci, bukan nilai. Ini sangat berguna untuk membuat program yang memeriksa ketersediaan data sebelum mencoba mengaksesnya, yang bisa mencegah KeyError.

Mengulang Item dalam Dictionary: Perulangan for

Sama seperti List dan Tuple, perulangan for juga sangat cocok untuk Dictionary. Anda bisa mengulang kuncinya, nilainya, atau keduanya sekaligus.

Mengulang Kunci (Cara Default)

Jika Anda hanya mengulang Dictionary langsung, for akan mengulang setiap kuncinya. 

profil_guru = {
    "nama_lengkap": "Pak Budi",
    "bidang": "Matematika",
    "usia": 40
}

print("--- Data Profil Guru ---")
for kunci in profil_guru: # Secara default, 'for' mengulang kuncinya
    nilai = profil_guru[kunci] # Ambil nilai menggunakan kunci
    print(f"{kunci.replace('_', ' ').title()}: {nilai}") # Manipulasi string untuk tampilan rapi

Outputnya:

--- Data Profil Guru ---
Nama Lengkap: Pak Budi
Bidang: Matematika
Usia: 40

Penjelasan Kode:

Pada for kunci in profil_guru:, variabel kunci akan berisi "nama", lalu "bidang", lalu "usia" secara berurutan. Di dalam perulangan, kita menggunakan profil_guru[kunci] untuk mendapatkan nilai yang bersesuaian. Disini terdapat tambahan manipulasi string (.replace('_', ' ').title()) agar kunci "nama_lengkap" misalnya, bisa tampil rapi menjadi "Nama Lengkap".

Mengulang Nilai

Jika Anda hanya tertarik pada nilai-nilainya, Anda bisa mengulang dictionary.values(). 

skor_siswa = {
    "Andi": 85,
    "Budi": 90,
    "Cici": 78,
    "Dina": 92
}
total_skor = 0

print("--- Daftar Skor Siswa ---")
for skor in skor_siswa.values(): # 'skor' akan berisi 85, lalu 90, dst.
    print(f"Skor: {skor}")
    total_skor += skor

rata_rata_skor = total_skor / len(skor_siswa)
print(f"Total Skor: {total_skor}")
print(f"Rata-rata Skor: {rata_rata_skor}")

Outputnya:

--- Daftar Skor Siswa ---
Skor: 85
Skor: 90
Skor: 78
Skor: 92
Total Skor: 345
Rata-rata Skor: 86.25

Penjelasan Kode:

Dengan for skor in skor_siswa.values():, variabel skor akan langsung berisi nilai-nilai (85, 90, 78, 92) tanpa perlu mengakses kunci terlebih dahulu.

Mengulang Kunci dan Nilai Sekaligus

Ini adalah cara paling umum dan efisien untuk mengulang Dictionary ketika Anda membutuhkan kedua-duanya, menggunakan dictionary.items(). 

ketersediaan_produk = {
    "Meja": 15,
    "Kursi": 20,
    "Lampu": 10,
    "Lemari": 5
}

print("--- Laporan Stok Produk ---")
for nama_produk, jumlah_stok in ketersediaan_produk.items():
    print(f"{nama_produk} : {jumlah_stok} unit")

print("\n")

print("--- Produk Hampir Habis ---")
for produk, stok in ketersediaan_produk.items():
    if stok < 10: # Kondisional di dalam perulangan
        print(f"Peringatan! {produk} tersisa {stok} unit.")

Outputnya:

--- Laporan Stok Produk ---
Meja : 15 unit
Kursi : 20 unit
Lampu : 10 unit
Lemari : 5 unit

--- Produk Hampir Habis ---
Peringatan! Lampu tersisa 10 unit.
Peringatan! Lemari tersisa 5 unit.

Penjelasan Kode:

Pada for nama_produk, jumlah_stok in ketersediaan_produk.items():, setiap pasangan kunci-nilai dari items() akan "dibongkar" (unpacking) secara otomatis ke dua variabel: nama_produk akan mendapatkan kunci, dan jumlah_stok akan mendapatkan nilainya. Ini adalah cara yang sangat rapi dan mudah dibaca untuk memproses setiap item di Dictionary.

Batasan pada Kunci (Keys) Dictionary

Kita sudah tahu bahwa kunci di Dictionary harus unik. Tapi ada batasan lain yang penting: Kunci Dictionary harus berupa tipe data yang tidak dapat diubah (immutable).

  • Bisa jadi kunci: Angka (int, float), string, dan Tuple.
  • Tidak bisa jadi kunci: List, Set, dan Dictionary lain.

Mengapa? Karena Dictionary menggunakan teknik khusus bernama "hashing" untuk menyimpan dan menemukan kunci dengan sangat cepat. Hanya tipe data yang tidak dapat diubah yang memiliki "hash" yang stabil. Jika kunci bisa berubah, hash-nya juga bisa berubah, dan Dictionary tidak akan bisa menemukan kunci tersebut lagi! 

# Contoh kunci yang BENAR:
dict_kunci_ok = {
    1: "Angka",
    "nama": "Teks",
    (1, 2): "Tuple" # Tuple bisa jadi kunci karena immutable
}
print("Dictionary dengan kunci yang valid:", dict_kunci_ok)

# Contoh kunci yang SALAH (akan error!)
list_sebagai_kunci = [1, 2]
# dict_kunci_error = {list_sebagai_kunci: "Ini error"} # Ini akan error! TypeError: unhashable type: 'list'
# print(dict_kunci_error)

set_sebagai_kunci = {10}
# dict_kunci_error_2 = {set_sebagai_kunci: "Ini juga error"} # Ini akan error! TypeError: unhashable type: 'set'
# print(dict_kunci_error_2)

Outputnya:

Dictionary dengan kunci yang valid: {1: 'Angka', 'nama': 'Teks', (1, 2): 'Tuple'}

Penjelasan Kode:

Contoh ini dengan jelas menunjukkan bahwa Anda bisa menggunakan angka, string, atau Tuple sebagai kunci. Namun, jika Anda mencoba menggunakan List atau Set (atau Dictionary lain) sebagai kunci, Python akan mengeluarkan TypeError, karena mereka adalah tipe data yang mutable (dapat diubah).

Kapan Menggunakan Dictionary?

Anda akan memilih Dictionary ketika:

  1. Anda ingin menyimpan data yang memiliki "nama" atau "label" unik. Contoh: data siswa (nama, umur, kelas), informasi produk (nama, harga, stok), pengaturan aplikasi.
  2. Anda perlu mengakses informasi dengan cepat berdasarkan nama/identitasnya, bukan berdasarkan urutan (indeks). Mencari nilai di Dictionary berdasarkan kunci sangat cepat, bahkan untuk Dictionary yang sangat besar.
  3. Jumlah item dalam kumpulan data Anda mungkin tidak tetap (akan sering ditambah, dihapus, atau diubah).
  4. Anda ingin merepresentasikan objek atau entitas dengan beberapa atribut. Misalnya, satu Dictionary bisa mewakili satu orang dengan kunci seperti "nama", "usia", "alamat".

Dictionary adalah alat yang sangat fundamental dan serbaguna dalam pemrograman Python. Anda akan sering menggunakannya untuk mengatur dan mengelola data yang terstruktur.
