Python Fundamental

"Modul & Library" – Menggunakan Kode yang Sudah Dibuat Orang Lain!

Bayangkan Anda ingin membangun sebuah rumah. Anda tidak perlu membuat setiap bata, setiap jendela, atau setiap pintu dari nol. Anda tinggal membeli bata dari pabrik, jendela dari toko, dan pintu dari pengrajin. Dengan begitu, Anda bisa fokus pada merancang dan merakit rumahnya.

Dalam pemrograman Python, konsep Modul dan Library (atau Paket) mirip dengan itu. Mereka adalah kumpulan kode yang sudah ditulis dan diorganisir oleh orang lain (atau Anda sendiri) untuk melakukan tugas-tugas tertentu. Dengan menggunakan modul dan library, kita tidak perlu "menciptakan roda dua kali". Kita bisa menggunakan fungsi, variabel, atau bahkan struktur data canggih yang sudah tersedia, sehingga kita bisa fokus pada logika unik program kita.

1. Apa itu "Modul"?

Sebuah Modul di Python adalah sebuah file (.py) yang berisi kode Python. Di dalamnya bisa ada variabel, fungsi, atau bahkan definisi struktur data yang sudah kita pelajari. Modul ini bisa kita "impor" ke dalam program kita dan menggunakan isi di dalamnya. Ini adalah cara Python untuk mengorganisir kode ke dalam file-file terpisah agar lebih rapi dan bisa digunakan kembali.

Menggunakan Modul Bawaan Python: math dan random

Python dilengkapi dengan banyak modul bawaan yang sangat berguna. Mari kita coba dua di antaranya:

Modul math: Untuk Operasi Matematika Canggih

Modul math menyediakan fungsi-fungsi matematika yang lebih kompleks daripada operator dasar (+, -, *, /). 

# Cara 1: Mengimpor seluruh modul
import math

print("--- Menggunakan Modul math ---")
jari_jari = 7
luas_lingkaran = math.pi * (jari_jari ** 2) # math.pi adalah nilai PI (3.14...)
print(f"Luas lingkaran dengan jari-jari {jari_jari}: {luas_lingkaran:.2f}")

akar_kuadrat = math.sqrt(25) # math.sqrt() untuk akar kuadrat
print(f"Akar kuadrat dari 25: {akar_kuadrat}")

derajat = 45
radian = math.radians(derajat) # Mengubah derajat ke radian
print(f"{derajat} derajat = {radian:.4f} radian")
print(f"Sinus {derajat} derajat:", math.sin(radian)) # math.sin() untuk sinus

# Cara 2: Mengimpor hanya bagian tertentu dari modul
from math import pi, sqrt, cos

print("\n--- Menggunakan Bagian Tertentu dari Modul math ---")
# Sekarang kita bisa langsung pakai pi dan sqrt tanpa 'math.'
luas_lingkaran_2 = pi * (8 ** 2)
print(f"Luas lingkaran dengan jari-jari 8: {luas_lingkaran_2:.2f}")

akar_kuadrat_lain = sqrt(100)
print(f"Akar kuadrat dari 100: {akar_kuadrat_lain}")

# Menggunakan cosinus
sudut_radian = math.radians(60) # math.radians tetap perlu karena tidak diimpor langsung
cosinus_60 = cos(sudut_radian)
print(f"Cosinus 60 derajat: {cosinus_60:.2f}")

Outputnya:

--- Menggunakan Modul math ---
Luas lingkaran dengan jari-jari 7: 153.94
Akar kuadrat dari 25: 5.0
45 derajat = 0.7854 radian
Sinus 45 derajat: 0.71

--- Menggunakan Bagian Tertentu dari Modul math ---
Luas lingkaran dengan jari-jari 8: 201.06
Akar kuadrat dari 100: 10.0
Cosinus 60 derajat: 0.50

Penjelasan Kode:

  • import math: Baris ini memberitahu Python untuk memuat seluruh modul math. Setelah itu, untuk menggunakan apapun dari modul math (misalnya pi atau sqrt), kita harus menuliskannya dengan math.nama_yang_diinginkan (contoh: math.pi, math.sqrt).
  • from math import pi, sqrt, cos: Baris ini lebih spesifik. Ia hanya mengimpor pi, sqrt, dan cos dari modul math. Setelah itu, kita bisa langsung menggunakan pi, sqrt, dan cos tanpa perlu menulis math.. Cara ini membuat kode lebih ringkas jika Anda hanya butuh beberapa hal dari sebuah modul.

Modul random: Membuat Angka Acak

Modul random menyediakan fungsi untuk menghasilkan angka acak, sangat berguna untuk permainan, simulasi, atau bahkan mengacak daftar. 

import random

print("--- Menggunakan Modul random ---")
# Mengacak angka bulat antara 1 dan 10 (inklusif)
dadu = random.randint(1, 6) # Angka acak dari 1 sampai 6 (seperti dadu)
print(f"Hasil lemparan dadu: {dadu}")

# Mengacak angka desimal antara 0.0 dan 1.0
probabilitas = random.random()
print(f"Angka acak antara 0.0 dan 1.0: {probabilitas:.4f}")

# Memilih item acak dari sebuah list
daftar_pemenang = ["Ani", "Budi", "Cici", "Dina", "Eko"]
pemenang_undian = random.choice(daftar_pemenang)
print(f"Pemenang undian adalah: {pemenang_undian}")

# Mengacak urutan list (shuffle)
urutan_tim = ["Tim A", "Tim B", "Tim C", "Tim D"]
print("Urutan tim awal:", urutan_tim)
random.shuffle(urutan_tim) # Mengubah urutan list secara langsung
print("Urutan tim setelah diacak:", urutan_tim)

Outputnya (output acak, bisa berbeda setiap dijalankan): 

--- Menggunakan Modul random ---
Hasil lemparan dadu: 3
Angka acak antara 0.0 dan 1.0: 0.7821
Pemenang undian adalah: Dina
Urutan tim awal: ['Tim A', 'Tim B', 'Tim C', 'Tim D']
Urutan tim setelah diacak: ['Tim C', 'Tim A', 'Tim D', 'Tim B']

Penjelasan Kode:

  • random.randint(1, 6) menghasilkan integer acak antara 1 dan 6 (termasuk 1 dan 6).
  • random.random() menghasilkan float acak antara 0.0 (termasuk) dan 1.0 (tidak termasuk).
  • random.choice() memilih satu item acak dari list yang diberikan.
  • random.shuffle() mengubah urutan item dalam list secara langsung (in-place shuffle). Ini tidak mengembalikan list baru.

Mengganti Nama Modul Saat Mengimpor (as)

Jika nama modul terlalu panjang atau ingin disingkat, Anda bisa menggunakan as saat mengimpornya. 

import math as mt # Sekarang kita bisa pakai 'mt' sebagai pengganti 'math'
import random as rnd # Sekarang kita bisa pakai 'rnd' sebagai pengganti 'random'

print(f"Nilai PI (dari mt): {mt.pi:.3f}")
print(f"Angka acak (dari rnd): {rnd.random():.3f}")

Outputnya:

Nilai PI (dari mt): 3.142
Angka acak (dari rnd): 0.123 (bisa berbeda)

Penjelasan Kode:

import math as mt membuat Anda bisa menggunakan mt daripada math di seluruh kode Anda, yang kadang lebih cepat diketik atau lebih sesuai dengan gaya Anda.

Mengimpor Semua dari Modul (*) (Tidak Disarankan!)

Anda bisa mengimpor semua yang ada di modul langsung ke program Anda dengan from modul import *. Namun, ini sangat tidak disarankan karena bisa menyebabkan "tabrakan nama" (name collisions) jika ada nama fungsi atau variabel yang sama di modul yang berbeda atau di kode Anda sendiri. 

# SANGAT TIDAK DISARANKAN dalam kode yang besar!
from math import * # Mengimpor semua fungsi/variabel dari math

print("--- Mengimpor Semua (*) ---")
print(f"Nilai PI (langsung): {pi:.3f}") # Tidak perlu 'math.'
print(f"Sinus 90 derajat: {sin(radians(90)):.3f}") # Tidak perlu 'math.'

Outputnya:

--- Mengimpor Semua (*) ---
Nilai PI (langsung): 3.142
Sinus 90 derajat: 1.000

Penjelasan Kode:

Meskipun ini membuat kode terlihat lebih ringkas, bayangkan jika Anda mengimpor dua modul yang keduanya punya fungsi bernama hitung(). Python tidak akan tahu hitung() mana yang Anda maksud! Jadi, hindari import * kecuali Anda benar-benar yakin dan tahu apa yang Anda lakukan (misalnya, untuk eksperimen cepat di konsol Python).

Membuat Modul Anda Sendiri

Kekuatan modul bukan hanya pada modul bawaan, tapi juga pada kemampuan Anda untuk membuat modul sendiri! Jika Anda menulis fungsi-fungsi yang Anda rasa akan sering Anda gunakan di berbagai program, Anda bisa menyimpannya dalam sebuah file .py terpisah, lalu mengimpornya ke program utama Anda.

Langkah-langkah:

  1. Buat file Python baru (misalnya, alat_bantu.py).
  2. Isi file tersebut dengan fungsi atau variabel yang ingin Anda jadikan modul.
  3. Simpan file tersebut di direktori yang sama dengan program utama Anda, atau di lokasi yang bisa ditemukan Python.
  4. Impor modul tersebut di program utama Anda.

Contoh:

File: alat_bantu.py 

# alat_bantu.py

def salam(nama):
    """Mengucapkan salam kepada seseorang."""
    return f"Halo, {nama}! Selamat belajar."

def hitung_rata_rata(daftar_angka):
    """Menghitung rata-rata dari daftar angka."""
    if not daftar_angka: # Kondisional: jika list kosong
        return 0
    total = sum(daftar_angka) # sum() adalah fungsi bawaan Python untuk menjumlahkan isi list
    return total / len(daftar_angka)

PHI_LOKAL = 3.14 # Variabel di dalam modul

# Bagian ini akan dijalankan HANYA jika file ini dijalankan langsung, BUKAN saat diimpor
if __name__ == "__main__":
    print("Saya sedang dijalankan langsung sebagai alat_bantu.py")
    print(salam("Pengguna Langsung"))
    print("Rata-rata dari [1,2,3]:", hitung_rata_rata([1,2,3]))

File: program_utama.py (simpan di direktori yang sama dengan alat_bantu.py) 

# program_utama.py

import alat_bantu

print("--- Menggunakan Modul Buatan Sendiri ---")
pesan_salam = alat_bantu.salam("Siswa SMA")
print(pesan_salam)

nilai_saya = [80, 90, 75, 88, 92]
rata_saya = alat_bantu.hitung_rata_rata(nilai_saya)
print(f"Rata-rata nilai saya: {rata_saya:.2f}")

print(f"Nilai PHI dari modul: {alat_bantu.PHI_LOKAL}")

# Kita juga bisa mengimpor spesifik
from alat_bantu import salam, PHI_LOKAL
print(salam("Budi"))
print(PHI_LOKAL * 2)

Outputnya (jika Anda menjalankan python program_utama.py): 

--- Menggunakan Modul Buatan Sendiri ---
Halo, Siswa SMA! Selamat belajar.
Rata-rata nilai saya: 85.00
Nilai PHI dari modul: 3.14
Halo, Budi! Selamat belajar.
6.28

(Jika Anda menjalankan python alat_bantu.py langsung, Anda akan melihat output dari blok if __name__ == "__main__":)

Penjelasan Kode:

  1. Di alat_bantu.py, kita mendefinisikan dua fungsi (salam, hitung_rata_rata) dan satu variabel (PHI_LOKAL).
  2. Baris if __name__ == "__main__": adalah trik standar di Python. Kode di dalam blok ini hanya akan dijalankan jika file alat_bantu.py dijalankan langsung (misalnya, dari Command Prompt/Terminal). Jika file ini diimpor oleh file lain (seperti program_utama.py), maka kode di dalam blok if __name__ == "__main__": ini tidak akan dijalankan. Ini sangat berguna untuk menaruh kode pengujian atau demo di dalam modul tanpa menjalankannya setiap kali modul diimpor.
  3. Di program_utama.py, import alat_bantu membuat semua yang ada di alat_bantu.py bisa diakses dengan awalan alat_bantu..
  4. from alat_bantu import salam, PHI_LOKAL memungkinkan kita menggunakan salam dan PHI_LOKAL secara langsung tanpa awalan alat_bantu..

2. Apa itu "Library" (atau "Paket")?

Sebuah Library (sering juga disebut Paket) adalah kumpulan dari beberapa modul yang saling terkait, diorganisir dalam sebuah folder. Misalnya, ada library untuk membuat grafik, untuk memproses gambar, atau untuk terhubung ke internet. Python memiliki ekosistem library yang sangat besar (disebut PyPI - Python Package Index) yang berisi ribuan library yang bisa Anda instal dan gunakan.

Contoh library:

  • numpy: Untuk perhitungan matematika dengan angka-angka besar (array).
  • pandas: Untuk mengolah data berbentuk tabel (seperti di Excel).
  • matplotlib: Untuk membuat grafik dan visualisasi data.
  • requests: Untuk mengambil data dari internet.

Anda biasanya menginstal library ini menggunakan alat yang disebut pip (akan dibahas lebih lanjut di modul Virtual Environment & Package Management). Setelah terinstal, Anda bisa mengimpor modul di dalamnya seperti biasa.

Kapan Menggunakan Modul/Library?

  • Untuk tugas berulang: Jika Anda melakukan hal yang sama berkali-kali di program Anda sendiri, jadikan fungsi dan taruh di modul.
  • Untuk tugas yang kompleks: Jika ada tugas yang sudah ada solusinya (misalnya, menghitung akar kuadrat, mengirim email, membuat grafik), cari apakah ada modul atau library yang sudah ada yang bisa melakukannya.
  • Untuk mengorganisir kode: Pisahkan bagian-bagian program Anda ke dalam file .py yang berbeda sebagai modul untuk menjaga kerapian.

Apa yang Harus Diperhatikan: Pencarian Modul oleh Python

Ketika Anda menulis import nama_modul, Python akan mencari file nama_modul.py di beberapa tempat, secara berurutan:

  1. Direktori tempat program utama Anda berjalan: Ini sebabnya program_utama.py bisa menemukan alat_bantu.py jika keduanya di folder yang sama.
  2. Direktori yang tercantum dalam variabel lingkungan PYTHONPATH: Ini adalah pengaturan tingkat sistem operasi yang bisa ditambahkan untuk memberitahu Python di mana lagi harus mencari modul.
  3. Direktori instalasi standar Python: Di sinilah modul-modul bawaan seperti math, random, dll., disimpan.

Anda bisa melihat daftar semua direktori tempat Python mencari modul dengan mencetak sys.path (Anda harus import sys terlebih dahulu). 

import sys

print("--- Direktori Pencarian Modul Python (sys.path) ---")
for path in sys.path:
    print(path)

Outputnya (bisa berbeda tergantung sistem operasi dan instalasi Python Anda): 

--- Direktori Pencarian Modul Python (sys.path) ---
/path/ke/folder/program_utama_anda
/path/ke/instalasi/python/lib/python3.x
/path/ke/instalasi/python/lib/python3.x/site-packages
...

Penjelasan Kode:

sys.path adalah sebuah list yang berisi semua folder tempat Python akan mencari file .py ketika Anda menggunakan import. Jika modul yang ingin Anda impor tidak ditemukan di salah satu jalur ini, Python akan memberikan ModuleNotFoundError. Memahami ini berguna untuk memecahkan masalah jika Python tidak bisa menemukan modul yang Anda buat atau instal.
