"Konversi Tipe Data" – Mengubah Jenis Data

Kita sudah belajar bahwa setiap informasi di Python punya "jenis" atau "tipe data" (seperti angka bulat int, angka desimal float, atau teks str). Kita juga tahu bahwa operator (seperti +) bisa memiliki perilaku yang berbeda tergantung pada tipe data yang dioperasikannya. Misalnya, 5 + 3 akan menjumlahkan, tapi "Halo" + "Dunia" akan menggabungkan teks.

Kadang-kadang, informasi yang kita miliki tidak sesuai dengan tipe data yang kita butuhkan. Misalnya, Anda punya angka 10 sebagai teks ("10"), tapi Anda ingin menjumlahkannya dengan angka lain. Atau, Anda punya angka desimal 3.14, tapi Anda hanya ingin bagian bulatnya 3. Di sinilah konsep Konversi Tipe Data berperan.

Konversi tipe data adalah proses mengubah satu jenis data menjadi jenis data yang lain. Python menyediakan "alat bantu" khusus berupa fungsi yang bisa kita gunakan untuk melakukan konversi ini secara sengaja.

Mengubah ke Bilangan Bulat: Fungsi int()

Jika Anda memiliki sebuah angka desimal (float) dan Anda hanya ingin mengambil bagian bilangan bulatnya, atau Anda memiliki sebuah teks (string) yang isinya adalah angka bulat, Anda bisa menggunakan fungsi int(). 

# Mengubah dari float ke int
tinggi_meter = 1.78
tinggi_bulat = int(tinggi_meter) # Mengambil bagian bulatnya saja (membulatkan ke bawah)

print("Tinggi dalam meter (asli):", tinggi_meter)
print("Tinggi setelah diubah ke bilangan bulat:", tinggi_bulat)

print("\n") # Baris kosong untuk keterbacaan

# Mengubah dari string ke int (jika isinya angka bulat)
jumlah_barang_text = "50"
jumlah_barang_angka = int(jumlah_barang_text)

print("Jumlah barang (sebagai teks):", jumlah_barang_text, type(jumlah_barang_text))
print("Jumlah barang (setelah diubah ke angka):", jumlah_barang_angka, type(jumlah_barang_angka))

# Sekarang kita bisa menjumlahkannya!
hasil_jumlah = jumlah_barang_angka + 10
print("Jumlah barang ditambah 10:", hasil_jumlah)

print("\n")

# Apa yang terjadi jika string tidak berisi angka bulat? ERROR!
teks_bukan_angka = "lima belas"
# angka_dari_teks = int(teks_bukan_angka) # Baris ini akan menyebabkan error!
# print(angka_dari_teks)

teks_desimal = "3.14"
# angka_dari_desimal_text = int(teks_desimal) # Baris ini juga akan menyebabkan error!
# print(angka_dari_desimal_text)

Outputnya:

Tinggi dalam meter (asli): 1.78
Tinggi setelah diubah ke bilangan bulat: 1

Jumlah barang (sebagai teks): 50 <class 'str'>
Jumlah barang (setelah diubah ke angka): 50 <class 'int'>
Jumlah barang ditambah 10: 60

Penjelasan Kode:

Ketika Anda menggunakan int(tinggi_meter), Python akan mengambil 1.78 dan hanya mempertahankan bagian bulatnya, yaitu 1. Ingat, int() akan membulatkan ke bawah, jadi 1.99 pun akan menjadi 1.

Ketika int("50") dijalankan, Python mengenali bahwa teks "50" benar-benar merepresentasikan angka bulat, sehingga akan mengubahnya menjadi angka 50 bertipe int. Setelah itu, kita bisa melakukan operasi matematika pada jumlah_barang_angka.

Namun, penting untuk diingat: Anda hanya bisa mengubah string ke int jika string itu benar-benar terlihat seperti angka bulat. Jika Anda mencoba mengubah "lima belas" atau "3.14" menjadi int, Python akan "bingung" dan mengeluarkan pesan kesalahan (disebut ValueError), karena tidak bisa menemukan angka bulat di dalamnya.

Mengubah ke Bilangan Desimal: Fungsi float()

Jika Anda memiliki sebuah angka bulat (int) dan Anda ingin menganggapnya sebagai angka desimal, atau Anda memiliki sebuah teks (string) yang isinya adalah angka desimal atau bulat, Anda bisa menggunakan fungsi float(). 

# Mengubah dari int ke float
umur = 17
umur_desimal = float(umur)

print("Umur (sebagai angka bulat):", umur, type(umur))
print("Umur (setelah diubah ke desimal):", umur_desimal, type(umur_desimal))

print("\n")

# Mengubah dari string ke float (bisa angka bulat atau desimal)
harga_text = "25000"
harga_float = float(harga_text)

berat_text = "60.5"
berat_float = float(berat_text)

print("Harga (sebagai teks):", harga_text, type(harga_text))
print("Harga (setelah diubah ke desimal):", harga_float, type(harga_float))

print("Berat (sebagai teks):", berat_text, type(berat_text))
print("Berat (setelah diubah ke desimal):", berat_float, type(berat_float))

# Sekarang kita bisa menghitung sesuatu dengan angka desimal
total_harga_diskon = harga_float * 0.9 # Diskon 10%
print("Total harga setelah diskon:", total_harga_diskon)

print("\n")

# Apa yang terjadi jika string tidak berisi angka? ERROR!
ukuran_invalid = "ukuran besar"
# ukuran_angka = float(ukuran_invalid) # Baris ini akan menyebabkan error!
# print(ukuran_angka)

Outputnya:

Umur (sebagai angka bulat): 17 <class 'int'>
Umur (setelah diubah ke desimal): 17.0 <class 'float'>

Harga (sebagai teks): 25000 <class 'str'>
Harga (setelah diubah ke desimal): 25000.0 <class 'float'>
Berat (sebagai teks): 60.5 <class 'str'>
Berat (setelah diubah ke desimal): 60.5 <class 'float'>
Total harga setelah diskon: 22500.0

Penjelasan Kode:

Ketika float(umur) dijalankan, Python mengubah angka bulat 17 menjadi 17.0, menjadikannya tipe float. Ini berguna jika Anda ingin melakukan perhitungan yang melibatkan presisi desimal.

Fungsi float() juga bisa mengubah string menjadi angka desimal, selama string itu berisi representasi angka yang valid, baik itu angka bulat ("25000") maupun angka desimal ("60.5"). Teks "25000" akan menjadi 25000.0 setelah diubah oleh float().

Sama seperti int(), float() akan menghasilkan error jika string yang Anda coba ubah tidak bisa dikenali sebagai angka.

Mengubah ke Teks: Fungsi str()

Mungkin ini yang paling sering Anda butuhkan! Jika Anda memiliki angka (int atau float) atau nilai Boolean (True/False) dan Anda ingin menampilkannya bersama dengan teks lain, cara terbaik adalah mengubahnya menjadi string terlebih dahulu. Python seringkali bisa melakukannya secara otomatis dalam perintah print(), tetapi ada kalanya Anda perlu melakukannya secara manual, terutama saat menggabungkan teks menggunakan operator +. 

# Mengubah dari int ke str
nilai = 90
pesan_nilai = "Nilai Anda adalah: " + str(nilai) # Mengubah 90 menjadi "90"

print(pesan_nilai, type(pesan_nilai))

print("\n")

# Mengubah dari float ke str
phi = 3.14159
teks_phi = "Nilai PI adalah: " + str(phi)

print(teks_phi, type(teks_phi))

print("\n")

# Mengubah dari bool ke str
is_aktif = True
status_akun = "Status akun: " + str(is_aktif)

print(status_akun, type(status_akun))

print("\n")

# Mengapa perlu str()?
# print("Total" + 100) # Ini akan ERROR!
print("Total " + str(100)) # Ini akan bekerja karena 100 sudah jadi teks "100"

Outputnya:

Nilai Anda adalah: 90 <class 'str'>

Nilai PI adalah: 3.14159 <class 'str'>

Status akun: True <class 'str'>

Total 100

Penjelasan Kode:

Fungsi str() adalah kebalikan dari int() atau float(). Ia mengambil nilai dari tipe data apapun (angka, boolean, dll.) dan mengubahnya menjadi representasi teksnya. Jadi, angka 90 akan menjadi string "90", 3.14159 menjadi "3.14159", dan True menjadi "True".

Ini sangat penting ketika Anda ingin menggabungkan teks dengan angka menggunakan operator +. Python tidak tahu bagaimana "menambahkan" teks "Total" dengan angka 100. Tapi, jika Anda mengubah 100 menjadi string "100" menggunakan str(100), maka Python bisa dengan mudah menggabungkan "Total " dengan "100" menjadi "Total 100".

Mengubah ke Boolean: Fungsi bool()

Fungsi bool() digunakan untuk mengubah nilai dari tipe data lain menjadi nilai Boolean (True atau False). Ini seringkali tidak disadari, tetapi ada aturan menarik tentang bagaimana Python menentukan apakah suatu nilai "benar" atau "salah" dalam konteks Boolean:

  • Angka: Angka 0 (nol) akan dianggap False. Semua angka lain (positif atau negatif, bulat atau desimal) akan dianggap True.
  • Teks (String): String kosong ("" atau '') akan dianggap False. Semua string lain yang berisi setidaknya satu karakter (termasuk spasi) akan dianggap True.
  • Nilai Khusus None: Ada nilai khusus di Python bernama None (yang berarti "tidak ada nilai"). None akan dianggap False. 
# Mengubah angka ke boolean
print("Angka 0:", bool(0))
print("Angka 1:", bool(1))
print("Angka -5:", bool(-5))
print("Angka 0.0:", bool(0.0))
print("Angka 3.14:", bool(3.14))

print("\n")

# Mengubah string ke boolean
print("String kosong (''):", bool(""))
print("String berisi ('Halo'):", bool("Halo"))
print("String berisi spasi (' '):", bool(" ")) # Spasi juga dianggap karakter!

print("\n")

# Mengubah nilai khusus None ke boolean
# (Kita akan belajar lebih banyak tentang None nanti)
data_kosong = None
print("Nilai None:", bool(data_kosong))

Outputnya:

Angka 0: False
Angka 1: True
Angka -5: True
Angka 0.0: False
Angka 3.14: True

String kosong (''): False
String berisi ('Halo'): True
String berisi spasi (' '): True

Nilai None: False

Penjelasan Kode:

Fungsi bool() sangat menarik karena ia menerapkan aturan "truthy" dan "falsy". Artinya, beberapa nilai secara inheren dianggap "benar" atau "salah" ketika diubah ke Boolean. Angka nol (0) dan string kosong ("") adalah contoh nilai "falsy" yang akan menjadi False saat dikonversi ke Boolean. Sementara itu, semua angka selain nol, dan semua string yang tidak kosong (bahkan hanya spasi), akan menjadi True ("truthy"). Konsep ini akan sangat berguna ketika Anda mulai membuat program yang membuat keputusan berdasarkan ada tidaknya data atau nilai.

Mengapa Konversi Itu Penting dan Sering Terjadi?

Konversi tipe data adalah salah satu hal yang paling sering Anda lakukan (secara sadar atau tidak) saat membuat program. Ini adalah jembatan yang memungkinkan Anda untuk:

  1. Melakukan Perhitungan: Jika Anda menerima input angka dari pengguna (yang seringkali datang sebagai string), Anda harus mengubahnya menjadi int atau float sebelum bisa menjumlahkan, mengurangi, dan sebagainya.
  2. Menampilkan Informasi: Jika Anda ingin menampilkan hasil perhitungan (angka) bersamaan dengan teks penjelasan, Anda mungkin perlu mengubah angka tersebut menjadi string.
  3. Memastikan Kompatibilitas: Memastikan bahwa data yang Anda gunakan dalam suatu operasi memiliki tipe yang sesuai.

Memahami kapan dan bagaimana melakukan konversi tipe data adalah kunci untuk menulis kode Python yang bekerja dengan benar dan efisien, serta menghindari kesalahan yang umum terjadi.
